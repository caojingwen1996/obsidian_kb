# 支柱产业自动化

本目录是“支柱产业”自动化任务的固定入口，即使暂时没有当日产物也应保留。

## 目录职责

- 保存支柱产业自动化的运行说明、产业级汇总、状态文件和必要的导航清单。
- 不保存具体标的的完整研报、资金面分析 HTML 或其他决策工件。
- 具体标的产物按证券代码保存到 `workbench/targets/<证券代码>-<公司简称>/`。
- 自动化需要引用已归档产物时，使用指向 Workbench 的链接或清单，不复制第二份正文。
