6. 资金分歧与传导链

按紫金矿业的拆法，神马电力可以拆成：

股价 = 企业基本面价值 + 电网建设 / 复合外绝缘替代价值 + 高成长 / 出海 / 题材投机溢价

主导链条

海外电网更新 + 特高压 + 复合外绝缘替代 -> 高毛利和高成长预期 -> 成长资金与题材资金推升估值 -> 估值高位遇到减持、现金流和回撤压力 -> 主力 / 游资流出 -> 散户承接但价格修复弱 -> 结构性流出

反向链条

如果 42—43 元缩量企稳 + 半年报确认海外收入、毛利率和现金流 + 主力资金转正或 ETF 承接出现 -> 投机溢价出清后，基本面价值和产业周期价值重新主导

谁在承接：目前可见的是散户资金净流入这一订单代理；不能写成长期资金或机构真实承接。潜在退出路径是高估值成长资金继续撤退、减持预期压制、弱反弹后短线资金轮动离场。
