# 支柱产业-高端制造自动化

本目录是“支柱产业-高端制造”自动化任务的固定入口，即使暂时没有当日产物也应保留。

## 目录职责

- 保存支柱产业中高端制造方向的运行说明、产业级汇总、状态文件、导航清单和需要保留在 `sources/` 的 HTML 输出。
- Markdown 权威研报、跟踪节点和后续可编辑正文继续保存到 `workbench/targets/`。
- 自动化需要引用已归档产物时，使用指向 Workbench 的链接或清单，不复制第二份正文。
