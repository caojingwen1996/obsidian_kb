# 有色拐点逻辑推导流程图

核心结论：有色真正的拐点，在美联储降息预期到来；降息预期取决于通胀预期何时下降。

## 一、当前宏观约束：通胀抬头 -> 降息被制约 -> 货币偏紧

### 石油危机冲击

- 通胀抬头是“明牌”
- 油价与通胀预期强相关

### 降息风险上升

- 若过早降息，可能快速推高通胀
- 因此当前不会轻易降息

### 政策倾向

- 降息延后
- 甚至存在加息概率

## 二、美联储先缩表的内在动机：减缓美债压力，而不是直接降息

### 财政部资金紧张

- 美债利息支出约 1.2 万亿美元
- 财政可支出空间被压缩
- 对应医保、精兵简政等压力

### 美联储回归货币

- 不直接替财政扩张兜底
- 通过货币端削弱基础货币供应速度
- 减缓利息支出继续膨胀

### 目标：软着陆

- 美债规模增长变慢
- 财政盈余后可偿还利息
- 美债不违约 -> 软着陆

### 关键观察点：6/16

需要等待缩表的路径、时间表、规模答案；因为缩表太快会引发市场振动，节奏非常重要。

## 三、缩表如何传导到黄金与有色：美元信用增强 -> 黄金承压 -> 有色被带崩

### 缩表规模与进程

- 接下来货币因子的核心变量
- 拉低基础货币供应量
- 影响资产定价

### 美债危机表现下降

- 美债规模增速放缓
- 美元信用加强
- 美元指数抬头 / 强美元诉求

### 黄金与有色承压

- 黄金资产 / 美债资产比值 < 1
- 黄金承压下跌
- 紫金矿业承压 -> 带崩有色

### 市场当前定价

市场已按“降息推迟、货币紧张”的逻辑推动有色下跌，股价已部分体现紧缩预期。

## 四、真正拐点：通胀预期下降 -> 降息预期升温 -> 有色见底

### 路径 A：石油危机转折

- 米伊局势出现结果
- 霍尔木兹海峡真正开通
- 油价 / 通胀预期下降

### 路径 B：中美贸易回暖

- 中国大量出口美国
- 消费品价格指数被打下来
- 通胀预期下降

### 关键变量：沃什理念

- 是否效仿格林斯潘
- 通过微观统计真实通胀
- 对通胀的容忍度很重要

### 最终推导：降息预期升温 -> 有色见底

若触发点来自中美贸易回暖，则中国经济基本面更强，股市走强概率增大；有色兼具指数力量，同步上涨。

## 图中结构

- 第一段：石油危机冲击 -> 降息风险上升 -> 政策倾向
- 第二段：财政部资金紧张 -> 美联储回归货币 -> 目标软着陆，并下接 6/16 观察点
- 第三段：缩表规模与进程 -> 美债危机表现下降 -> 黄金与有色承压，并与“市场当前定价”相连
- 第四段：路径 A、路径 B、沃什理念共同指向最终推导
