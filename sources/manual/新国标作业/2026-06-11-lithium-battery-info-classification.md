# 2026-06-11 锂电池信息分类日报

## 一、一句话结论

2026-06-11 是 7月1日新国标交易线的**前置预热日**：政策端已经由《经济参考报》明确传播“两项电动汽车安全新国标将落地”，监管端又出现工信部、市场监管总局约谈车企、要求强化产品质量管控，价格端则由碳酸锂现货与期货同步上涨带动锂矿股反弹。

这一天的核心价值在于：**新国标安全叙事开始被官方媒体提前传播，锂价和锂矿股先于电池安全链启动，说明资金最先交易的是材料价格弹性，后续需要观察能否切换到安全标准升级对应的检测、BMS、高压断电、热管理、结构件。**

当日评级：**B+（政策预热 + 价格确认 + 股价反应，尚缺新国标直接订单/认证公告）**。

## 四、重点信息拆解

### 1. 《经济参考报》：电动汽车两项新国标将落地

- **信息来源**：新华网 / 经济参考报
- **发布时间**：2026-06-11 10:37
- **原始事实**：文章称，7月1日起，《电动汽车安全要求》（GB 18384-2025）和《电动汽车用动力蓄电池安全要求》（GB 38031-2025）将正式实施；两项新规分别聚焦整车和动力电池安全，构筑电动汽车安全“双重防线”。
- **对应框架**：二策、四界
- **对应产业链环节**：动力电池、整车高压安全、检测认证、BMS、热管理、结构防护。
- **与新国标的关系**：直接相关，是6.11当天最强新国标信息。
- **交易含义**：政策信息从标准文件进入财经媒体传播，说明市场开始具备交易共识基础。
- **风险点**：6.11仍以传播为主，缺少具体企业认证、订单、财报验证。

### 2. 工信部、市场监管总局约谈涉嫌非理性竞争车企

- **信息来源**：新华社 / 新华网
- **发布时间**：2026-06-11 20:06
- **原始事实**：工信部、市场监管总局对涉嫌存在非理性竞争行为的汽车生产企业进行约谈提醒，要求加强价格合规建设、强化产品质量管控，维护优质优价和良性竞争秩序。
- **对应框架**：二策、五略
- **对应产业链环节**：整车、动力电池、上游供应链。
- **与新国标的关系**：间接相关。它不是电池标准本身，但强化“质量、安全、合规”权重。
- **交易含义**：整车价格战被监管约束后，低价压供应链的逻辑可能边际削弱，具备质量和安全优势的供应商更容易获得估值支持。
- **风险点**：该信息偏汽车行业监管，不直接带来锂电订单。

### 3. 碳酸锂现货与期货同步上涨

- **信息来源**：财联社 / Mysteel / SMM / 每日经济新闻
- **发布时间**：2026-06-11 10:33、13:31、15:59
- **原始事实**：MMLC 电池级碳酸锂早盘价格较前一日上涨950元/吨，中间价166600元/吨；碳酸锂期货主力合约日内涨超4%，报173900元/吨；广期所碳酸锂期货仓单54683手，环比减少898手。
- **对应框架**：三业、五略
- **对应产业链环节**：锂矿、锂盐、正极材料、储能电池。
- **与新国标的关系**：间接相关。价格上涨不直接来自新国标，但会增强材料链风险偏好。
- **交易含义**：6.11资金最先选择上游锂价弹性，锂矿是当天股价确认最强方向。
- **风险点**：锂价上涨对材料企业可能构成成本压力，是否能传导到利润需要财报验证。

### 4. 格林美共建固态锂电正极材料联合实验室

- **信息来源**：证券时报 / 人民财讯
- **发布时间**：2026-06-11 08:28
- **原始事实**：格林美表示，公司与宁波东方理工大学签署合作协议，共建固态锂电正极材料联合实验室，聚焦固态电池正极材料、电极界面改性、固态模具电池组装及固态电池循环再生等关键技术。
- **对应框架**：三业、五略
- **对应产业链环节**：正极材料、固态电池、回收。
- **与新国标的关系**：弱直接关系，强中长期技术路线关系。
- **交易含义**：固态电池和正极材料的技术叙事继续升温，但短期交易强度低于锂价和新国标安全链。
- **风险点**：实验室合作距离订单和业绩较远。

### 5. 石大胜华加码特种添加剂业务

- **信息来源**：证券时报 / 人民财讯
- **发布时间**：2026-06-11 19:12、21:04
- **原始事实**：石大胜华拟设立全资子公司，负责东营基地特种添加剂产品研发、生产及销售，注册资本金4000万元；公司称在锂电行业深耕20余年，具备五大碳酸酯溶剂、六氟磷酸锂及部分添加剂的生产能力。
- **对应框架**：三业
- **对应产业链环节**：电解液、添加剂、锂电材料。
- **与新国标的关系**：间接相关。电解液添加剂会影响电池性能、安全与循环表现，但公告没有直接指向新国标认证。
- **交易含义**：电解液添加剂方向在材料链景气中具有弹性，后续需看价格和客户。
- **风险点**：子公司设立本身不等于订单增量。

## 五、冰冰小美式推导链

6.11 的信息从“一国”开始：新能源汽车渗透率继续提高，安全问题已经从局部事故变成产业长期约束。进入“二策”层，两项电动汽车安全新国标将在7月1日实施，同时监管约谈车企价格战，要求强化质量管控，这说明国家层面在把汽车行业从规模扩张引向质量、安全和合规。落到“三业”层，企业端出现格林美固态锂电正极材料联合实验室、石大胜华特种添加剂子公司，价格端出现碳酸锂现货和期货同步上涨，说明材料链先获得反馈。进入“四界”层，媒体借新国标和热失控舆情反复强化电池安全。最后到“五略”，资金在6.11选择锂矿和锂盐价格弹性先动，后面能否演化为新国标安全链，还要看检测设备、BMS、高压断电、热管理和结构件是否接力。

这一天支持“新国标改变锂电池生态”的程度为**中强**。强验证来自政策传播和监管质量约束；中等验证来自碳酸锂价格和锂矿股上涨；弱验证来自企业技术合作和舆情科普。真正缺失的是：企业通过新国标测试、订单公告、客户绑定、财报利润验证。

## 六、后续跟踪清单

1. **政策传播**：6.12 是否继续出现官方、应急管理、消防、行业媒体对 GB 18384-2025、GB 38031-2025 的传播。
2. **监管延伸**：约谈车企后，是否出现反内卷、质量抽查、产品一致性监督检查等后续动作。
3. **锂价延续**：碳酸锂期货是否站稳17万元/吨，现货价格是否继续跟涨。
4. **仓单变化**：广期所碳酸锂仓单是否继续减少，库存去化是否持续。
5. **材料链股价**：锂矿、锂盐上涨后，是否扩散到正极、电解液、铜箔、隔膜。
6. **安全链启动**：检测设备、BMS、高压断电、热管理、泄压阀、结构件是否出现补涨。
7. **企业公告**：是否出现“通过 GB 38031-2025 测试”“取得检测报告”“进入头部电池厂/车企供应链”。
8. **舆情催化**：热失控、起火、碰撞、底盘刮底等舆情是否继续强化电池安全关注。
9. **中报前置信号**：锂电材料企业是否发布订单增长、价格上调、产能利用率提升等信号。
10. **资金选择**：资金继续做锂矿价格弹性，还是切换到新国标直接映射的安全部件。

## 七、当日结论评级

**评级：B+**

### 评级理由

- **二策强**：两项新国标由财经主流媒体集中传播，监管同时约谈车企，质量与安全权重上升。
- **三业中强**：碳酸锂现货和期货同步上涨，材料链有价格验证；格林美、石大胜华提供企业端补充信息。
- **五略中强**：股价已经反应，但主要集中在锂矿和锂盐，上游弹性强于安全链。
- **短板**：没有发现当天企业直接公告通过新国标测试、认证、订单或客户绑定；安全链尚未成为资金主攻方向。

### 6.11 对后续的定位

6.11 更像是**新国标主线的预热启动日**。它把政策传播、质量监管、锂价反弹、锂矿股异动连接到一起，但还没有进入 6.12 之后的公告验证和 6.16 的资金共振阶段。
