## 原文内容

用别人的财报，验证目标公司的需求。

这套方法的本质是：用已经公开的财报证据，提前判断尚未披露业绩公司的需求拐点。

它包含三个角色：

1. 需求方向：市场正在交易的方向，例如高速铜箔、功率半导体、AI PCB、新能源车安全链、储能、机器人。
2. 旁证公司：已经披露财报，并且财报里可能暴露需求变化的公司，例如西部矿业、长电科技、PCB 厂、封测厂、材料厂、设备厂、上游资源厂。
3. 目标公司：真正想推导的标的，例如铜冠铜箔、新洁能，或者某个还没出中报的弹性标的。

方法公式：

```text
目标公司机会
=
需求方向
× 旁证公司财报异常
× 产业链对应关系
× 目标公司尚未充分反应
```

更口语化：

```text
需求有没有？
看旁边公司的财报。

目标公司能不能涨？
看这个需求能不能传导到它身上。

还能不能买？
看股价有没有提前涨完。
```

标准推导模板：

1. 市场在交易什么需求？
2. 这个需求会影响产业链哪些公司？
3. 哪家公司先披露财报？
4. 财报有没有异常增长？
5. 这个增长能否归因到该需求？
6. 如果能归因，目标公司的需求就被旁证验证。
7. 如果目标公司还没充分上涨，就存在时间差机会。
8. 后续用目标公司自己的财报做最终验证。

西部矿业与铜冠铜箔案例：

```text
AI 算力增长
-> 高速 PCB 需求增长
-> 高速铜箔需求增长
-> 西部矿业一季报大增
-> 铜相关需求和价格出现强化
-> 高速铜箔需求可能正在被真实采购验证
-> 铜冠铜箔从横盘进入启动
```

这条逻辑的核心是：铜矿财报先给出需求旁证，铜箔股后面交易需求确认。

长电科技与新洁能案例：

```text
新能源车 / 储能 / 光伏逆变器 / AI 电源 / 工业控制
-> 功率半导体需求增长
-> 长电科技财报改善
-> 如果改善来自功率器件、汽车电子、工业控制相关封测
-> 功率半导体出货需求在恢复
-> 新洁能的需求拐点获得旁证
```

这条逻辑的核心是：封测财报先给出出货旁证，功率半导体公司后面交易需求修复。

这套方法真正抓的是财报披露时间差：

```text
谁先出财报
-> 谁先暴露需求
-> 谁还没被市场充分定价
-> 谁后面可能被中报验证
```

三个命名层级：

- 最准确的名字：产业链财报旁证法。
- 更口语的名字：用别人的财报验证目标公司。
- 更偏交易的名字：财报时间差交易法。

最终定义：

通过产业链上已经披露财报的公司，寻找收入、利润、毛利率、订单、库存、稼动率等异常变化，再反向验证目标公司的需求是否真实发生，并利用财报披露节奏和市场反应差寻找交易机会。

核心问题：

1. 谁先披露财报？
2. 财报里有没有异常增长？
3. 异常增长来自哪个需求方向？
4. 这个需求能否传导到目标公司？
5. 目标公司股价有没有充分反应？
6. 后续用什么财报继续验证？

一句话版本：

产业链财报旁证法，就是用已经出财报的公司，提前验证还没出财报公司的需求拐点。

## 原文结构速览

- 核心定义：用已披露财报的产业链公司，旁证尚未披露业绩目标公司的需求拐点。
- 三个角色：需求方向、旁证公司、目标公司。
- 方法公式：目标公司机会 = 需求方向 × 旁证公司财报异常 × 产业链对应关系 × 目标公司尚未充分反应。
- 核心时间差：财报披露时间差与股价反应差。
- 案例线索：西部矿业 / 铜冠铜箔，长电科技 / 新洁能。
