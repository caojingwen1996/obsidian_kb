# 闵行一霸 - 2026-04-07 观点整理

## 总观点
作者当天围绕泡泡玛特与 IP 商品化表达了偏正面的判断：优质自有 IP 的持续打造非常困难，因此泡泡玛特能够高频跑出爆款 IP 属于稀缺能力；从消费端看，IP 溢价只有落在本来就有功能需求的产品上，才更容易成立。

## 分观点
1. 自有 IP 不是有渠道就能做出来，Top Toy 招股书反映出自有 IP 数量虽增加，但销售规模仍不高。

2. 优质 IP 的持续产出极其稀缺，连三丽鸥十年都未必能再出重磅 IP，泡泡玛特一年火一个 IP 的速度很快。

3. 消费者愿意为 IP 和设计支付一定溢价，但前提通常是商品本身也满足原有的功能需求，而不是单纯为热度买单。
