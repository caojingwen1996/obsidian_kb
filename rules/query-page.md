#  query page 

## 适用场景
Query Page = 知识库的问题引擎。

多个页面之间需要综合，主要回答：

- 这个问题为什么重要？
- 它从哪些页面中产生？
- 当前已有线索是什么？
- 还缺哪些证据？
- 解决后应该更新哪些页面？
- 这个问题是否需要认知加工？
