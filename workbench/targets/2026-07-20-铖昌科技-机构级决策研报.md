# 铖昌科技机构级决策研报

> 证券：浙江铖昌科技股份有限公司（001270.SZ）  
> 研究截止：2026-07-20 收盘（财务以已披露公告为准）  
> 当前价格：98.95 元；总市值：约 203.95 亿元  
> 研究目的：回答“如果今天持有现金，是否愿意买入铖昌科技”。

## 1. 决策摘要

| 决策字段 | 结论 |
|---|---|
| 估值状态 | **高估**。当前市值对应约154倍滚动市盈率、约118倍光大证券2026年盈利预测市盈率，已提前计入数年高增长 |
| 综合公允价值 | **45—70元/股**；对应权益价值约93—144亿元 |
| 相对现价空间 | 相对98.95元仍有约 **-55%至-29%** 的估值回归空间 |
| 冰冰小美动作 | 新资金：**observe / wait（观察 / 等待）**；已有持仓：**review（复盘）**，反弹但预期未上修时评估 reduce |
| 风险方向 | **风险重新增强**，置信度高；基本面风险边际减弱，但估值、杠杆与情绪负反馈重新占主导 |
| 三要素 | 竞争格局有利、流动性不利、情绪不利，**尚未形成有利共振** |
| 现金回答 | **今天不会买入，也不把连续跌停直接定义为“错杀”。** 先等待流动性止跌和中报现金流验证 |
| 结论置信度 | 中高。公告和财务证据充分；客户订单金额、可比估值口径和实时ETF申赎仍不完整 |

铖昌科技是国内相控阵T/R芯片的重要设计企业，星载、机载和地面应用构成较高技术壁垒。2025年收入接近翻倍、盈利恢复，2026Q1净利润继续增长50%，说明产业逻辑没有被最新财务证伪。但投资结论取决于“产业资格是否已被价格过度透支”：公司近五年自由现金流近似值持续为负，2026Q1末应收账款及票据高达7.52亿元，而当前股价即使按券商2028年3.27亿元净利润预测仍约62倍市盈率。基本面好转是真，当前赔率不足也是真。

## 4. 商业模式与竞争格局

铖昌科技采用Fabless模式，核心价值在相控阵T/R芯片设计、工艺适配、可靠性验证和客户定制。产品涵盖GaAs/GaN功率放大器、低噪声放大器、模拟波束赋形及控制等芯片，主要应用于星载、机载、地面相控阵雷达和低轨卫星通信。

### 比较优势

- **技术和认证壁垒：** 高频、高集成、轻量化和抗辐照要求高，产品从研发到客户定型具有较长验证周期；
- **完整产品矩阵：** 公司称已有上千种产品，可提供T/R芯片完整解决方案，减少客户多供应商适配成本；
- **星载先发与场景扩张：** 低轨卫星产品自2022年起批量交付，机载业务2025年成为主要收入来源之一，应用不再只有单一星载场景；
- **高研发强度：** 2025年研发费用1.46亿元，占收入36.08%，且全部费用化，利润没有依赖研发资本化美化。

### 约束与脆弱点

- 2025年前五大客户收入占94.36%，前两大客户合计70.50%，单一项目节奏会显著影响年度业绩；
- 前五大供应商采购占85.45%，晶圆代工等供应集中，议价和交付受上游约束；
- 客户回款周期通常超过一年，而供应商约一个月结算，形成结构性营运资金错配；
- 核心业务几乎全部来自T/R芯片，收入结构高度单一；
- 军工和航天项目存在招投标、验收、交付节奏波动，2024年的项目延后已证明这一风险可以直接冲击利润。

**竞争格局判断：有利，置信度中等。** 公司在细分技术、认证和客户关系上有比较优势，但极高客户与供应商集中度降低了优势兑现的稳定性。这里的“有利”指产业位置，不等于当前股票价格有利。

### 财务质量结论

1. 2021—2025年收入复合增速约17.7%，但年度波动大，2024年项目延期就使收入下降26.4%、利润转负；
2. 五年自由现金流近似值连续为负，说明会计利润尚未稳定转化为股东可支配现金；
3. 2025年研发费用1.460亿元，同比增长66.14%，占收入36.08%，投入强度高；
4. 2025年末货币资金3.004亿元且无有息借款，偿债风险低；核心矛盾是回款效率而不是债务压力；
5. 2025年确认规模效应和产品结构带来的毛利率修复，但持续性需等待至少两个季度验证。

## 9. 多头与空头论据

### 多头论据

- 低轨卫星组网、星载相控阵和机载雷达需求提供多年成长空间；
- 公司具备完整T/R芯片产品矩阵和批量交付经验，技术与认证壁垒高；
- 2025年和2026Q1盈利修复已由财报确认，毛利率回到高位；
- 无有息借款、现金类资产较充足，短期偿债风险小；
- ST撤销后制度性风险下降，研发投入为后续新品提供储备。

### 空头论据

- 当前估值提前透支至2028年，任何订单或验收延后都可能压缩估值；
- 五年自由现金流近似值连续为负，2026Q1应收账款及票据达到7.52亿元；
- 客户和供应商集中度极高，收入和毛利率难以按普通成长股线性外推；
- 2024年已发生项目延后、收入下降和信用减值，业务节奏风险不是假设；
- 7月14日龙虎榜净卖出约0.96亿元，而融资余额在下跌初期增至约2.66亿元，杠杆承接可能放大后续出清；
- 7月16日公司明确不存在应披露未披露重大事项，短期缺少能解释前期涨幅的新信息。

### 证伪条件

- **多头证伪：** 半年报经营现金流未转正、应收继续显著快于收入、毛利率回落至70%以下，或公司/券商连续下修2026—2027年利润；
- **空头证伪：** 半年报经营现金流显著转正、回款和收入同步改善，披露可量化大额订单，且股价在放量换手后不再创新低。

## 10. 估值

### 10.1 相对市盈率法

直接可比公司臻镭科技、雷电微力、国博电子在业务结构、盈利周期和会计利润上差异较大，2026-07-20动态市盈率分别约112倍、为负和失真，因此不能简单取均值。本文以铖昌科技自身增长与风险给出55—70倍的2026年合理市盈率，低于当前118倍，但仍显著高于普通军工电子公司。

- 2026年利润情景：1.73—2.20亿元；
- 合理权益价值：约95—154亿元；
- 对应每股价值：约46—75元。

### 10.2 DCF法

DCF以2025年收入4.046亿元为起点，净现金采用2.5亿元（保留经营安全垫），总股本2.061亿股。因历史自由现金流持续为负，结果对终值假设极敏感：

| 情景 | 2026—2030收入增速 | 2030收入 | 2030自由现金流率 | WACC / 永续增速 | 每股价值 |
|---|---|---:|---:|---:|---:|
| 悲观 | 10%、10%、8%、8%、5% | 6.00亿元 | 10% | 13% / 2% | 2.96元 |
| 基准 | 35%、30%、25%、20%、15% | 12.25亿元 | 23% | 11.5% / 3% | 12.97元 |
| 乐观 | 55%、40%、30%、25%、20% | 17.12亿元 | 32% | 10.5% / 3.5% | 29.55元 |

DCF不适合作为短期目标价，但清晰揭示：即便乐观假设，现金流折现价值也显著低于现价。反向看，当前约201亿元企业价值需要远高于上述乐观情景的长期收入或现金利润，估值安全边际不足。

### 10.3 行业特定EV/Sales法

对尚处高研发、回款滞后的卫星射频芯片企业，EV/Sales可作为成长权利的辅助口径：

- 2026年收入情景：5.4—6.2亿元；
- 合理EV/Sales：18—24倍；
- 加回调整后净现金2.5亿元；
- 对应权益价值约100—151亿元，每股约48—73元。

这一倍数已经包含稀缺性和商业航天成长溢价。若现金流不改善或订单延后，18倍也可能过高；若取得可验证的大额星座订单并稳定转化为现金，才有理由提高倍数。

### 10.4 交叉验证与公允价值

相对市盈率和EV/Sales交叉区间集中在46—75元，DCF为13—30元的风险锚。考虑公司技术稀缺性、资产负债表安全与商业航天成长权利，同时对客户集中和现金流给予折价，综合公允价值取 **45—70元/股**。这不是精确点位，而是需要用半年报不断更新的决策区间。

| 价格区间 | 估值含义 | 动作 |
|---|---|---|
| 高于80元 | 仍要求极高增长，安全边际不足 | 新资金等待；已有持仓复盘仓位 |
| 60—80元 | 进入估值观察区，仍需现金流确认 | 小仓观察，不因单日反弹追价 |
| 45—60元 | 接近合理区间 | 仅在现金流与趋势止跌共振时分批研究性买入 |
| 低于45元 | 具备价格安全边际的可能 | 先检查是否发生基本面证伪，再决定买入 |

## 11. 冰冰小美框架判断

### 11.1 三要素

| 维度 | 状态 | 核心证据 | 置信度 |
|---|---|---|---|
| 竞争格局 | **有利** | 技术认证壁垒、完整T/R芯片矩阵、星载先发与机载扩张；客户/供应商集中削弱稳定性 | 中 |
| 流动性 | **不利** | 宏观货币偏宽但信用扩张不强；个股高换手连续跌停、融资余额逆势上升，退出流动性恶化 | 高 |
| 情绪位置 | **不利** | 从商业航天拥挤上涨快速切换为连续跌停，负反馈尚未结束 | 高 |
| 共振 | **尚未形成有利共振** | 产业竞争优势没有转化为资金和情绪确认 | 高 |

宏观层面，6月末M2同比增长8%、M1增长4%、社融存量增长7.4%，1年期LPR为3.0%，人民币流动性并不紧；但上半年社融增量同比少2.02万亿元，说明宽货币没有形成同等强度的信用扩张。外部方面，美国10年期国债收益率4.55%，对长久期高估值资产仍是折现约束。资金来源更像主题轮动和杠杆交易，而不是可确认的持续产业增量资金。

### 11.2 风险方向

**最终方向：风险重新增强，置信度高。**

风险传导链为：

> 商业航天主题升温与ST撤销 → 估值和融资筹码快速拥挤 → 没有新增公告级基本面信息支撑高点 → 股价连续跌停、龙虎榜净卖出 → 杠杆承接面临去化 → 流动性与情绪负反馈重新增强。

这不是“公司基本面被完全证伪”：2025年和2026Q1仍显示盈利修复。它也不是纯粹“系统性恐慌”：7月20日市场虽偏弱，但个股下跌强度显著更大。更准确的归类是 **高估值、拥挤筹码与项目不确定性共同触发的个股再定价**。

### 11.3 动作映射

- **新资金：observe / wait。** 不在连续跌停阶段仅凭跌幅抄底；等待至少3—5个交易日不再创新低、成交额收敛且价格重新站稳关键均线；
- **已有轻仓：review。** 先确认买入逻辑是产业成长还是短线主题；若逻辑仍在，可等流动性恢复后处理，不把“不能卖出”误解为“估值合理”；
- **已有重仓：reduce优先级上升。** 若反弹至80元以上但盈利预测和现金流没有同步上修，应评估降低集中度；
- **重新进入买入评估：** 价格进入45—70元区间，同时半年报经营现金流转正、应收增速低于收入增速、无盈利预测下修，三项至少满足两项。

## 12. 最终结论

铖昌科技属于 **好赛道、强细分位置、但当前价格仍不便宜** 的公司。2025年和2026Q1财务修复证明产业逻辑仍在，ST撤销也消除了制度性风险；但客户高度集中、回款周期长、五年自由现金流持续为负，使其不能仅按“低轨卫星高景气”线性外推。

以98.95元和203.95亿元市值计，市场已经把2028年前后的利润扩张提前定价。综合估值45—70元，现价仍缺少安全边际。**如果今天持有现金，我选择观察，不买入；如果已经持有，我会按仓位大小复盘，并把半年报经营现金流作为去留的第一验证项。**

评级语言：**高估 / 观察**。不是做空建议，也不是断言股价不会反弹；它表达的是当前胜率、赔率和退出流动性没有同时满足。

## 13. 后续监控指标

| 指标 | 正面确认 | 风险触发 | 对应动作 |
|---|---|---|---|
| 2026H1经营现金流 | 明显转正，兑现管理层预期 | 仍为负或改善很弱 | 转正才进入买点评估；未兑现则下调估值 |
| 应收账款及票据 | 增速低于收入，回款明显 | 继续快于收入增长 | 风险触发时降低盈利质量评分 |
| 半年报归母净利润 | 约0.85—1.0亿元或更高 | 明显低于0.8亿元 | 低于阈值复核2026E预测 |
| 毛利率 | 维持75%以上 | 回落至70%以下 | 回落则下调PE和EV/Sales倍数 |
| 可量化订单 | 披露大额订单、客户验收或批量交付 | 只有产业新闻，无公司量化信息 | 只对公告级证据上调预期 |
| 存货与产销率 | 库存平稳、销售跟上产量 | 库存明显上升、交付延后 | 检查2024年风险是否重演 |
| 融资余额 | 随价格企稳温和下降 | 下跌中继续快速增加 | 后者视为杠杆风险继续增强 |
| 价格与成交 | 3—5日不创新低、缩量止跌后放量确认 | 再次跌停或反弹无量 | 不抢反弹；等待趋势确认 |

## 14. 未获取到的数据与研究局限

- 公司没有披露可验证的在手订单总额、订单交付日历和主要客户名称；
- 未取得完整四家机构一致预期明细，只核验了光大证券最新摘要，不能把单一券商预测当市场共识；
- 军工、星载和机载收入没有按场景完整分拆，无法做严格SOTP；
- 未取得商业航天主题ETF的当日申赎、机构逐笔成交和融资账户强平数据；
- 可比公司盈利口径差异大，动态市盈率存在负值和极端值，横向估值只作为边界参考；
- DCF对远期收入、现金流率和折现率极敏感，而公司历史自由现金流为负，结果只能作为风险锚；
- 行情数据来自公开聚合接口，最终成交统计以交易所为准。

这些缺口不会改变“现价安全边际不足”的主结论，但会影响合理区间的精度。取得半年报后应整体刷新财务、估值和三要素判断。
