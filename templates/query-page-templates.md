# {{title}}

## 1. 原始问题

> 在这里记录当时真正提出的问题，尽量保留原话。

示例：

> 为什么作者说 AI 产业趋势已经不可阻挡？

---

## 2. 提问背景

这个问题是在什么场景下提出的？

- 来自哪篇文章 / 哪个材料 / 哪次对话？
- 当时不理解的点是什么？
- 这个问题为什么值得单独记录？

---

## 3. 核心答案

用几句话记录当前得到的核心答案。

不要写太长，重点是保留结论。

```text
一句话答案：
```

---

## 4. 推导逻辑

记录答案是如何得出来的。

可以用简单链条表示：

```text
前提一：
前提二：
关键判断：
所以得出：
```

也可以写成：

```text
材料现象
-> 作者判断
-> 背后逻辑
-> 可沉淀的知识点
```
