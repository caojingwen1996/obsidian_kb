# 冰冰小美-非金融信息类型字典

## 1. 文档职责

本文件只负责定义“非金融信息类型”，并按照作者“非金融信息与有形之手接近”的原则给出参考级别。

本字典只回答：

1. 这条信息属于什么类型的非金融信息；
2. 它离“有形之手”有多近；
3. 它处在目标、路径、承接、传播、谋略中的哪一层；
4. 按作者框架，它的参考优先级有多高；
5. 它的识别边界是什么。

本字典不负责：

1. 判断是否进入某个 Step；
2. 判断风险等级；
3. 判断价格是否已经反应；
4. 判断配置、交易、跟踪、归档或回避；
5. 直接给出投资结论。

---

## 2. 核心原则

### 2.1 非金融信息的第一标准：是否贴近有形之手

金融信息的归纳与“钱”接近。  
非金融信息的归纳与“有形之手”接近。

非金融信息体现的是：

> 如何达到目标的过程。

因此，非金融信息的判断重点应放在目标与达成路径上，需要回答：

- 国家处在什么阶段；
- 有形之手想达成什么目标；
- 国策指向什么长期方向；
- 政策服务哪个阶段性目标；
- 行业、产业、企业是否承接这个目标；
- 舆论、评论、新闻、吃瓜是否在影响认知；
- 战略、方略、策略、计略、谋略分别处在什么层级。

## 4. 作者参考级别

| 参考级别 | 含义 | 判断标准 |
|---|---|---|
| P0 核心底层 | 最高优先级 | 国情国运、国策、长期规划、国家阶段性使命 |
| P1 高参考 | 高优先级 | 政策、五年规划执行、战略方向、阶段目标 |
| P2 承接验证 | 中高参考 | 行业、产业、企业是否承接国策政策 |
| P3 传播参考 | 低到中参考 | 评论、分析、新闻、吃瓜、市场认知变化 |
| PX 降权 / 过滤 | 低参考 | 情绪化、无依据、概念包装、立场诱导、吃瓜噪音 |

---

## 4.1 作者对不同非金融信息类型的态度总表

| 类型 | 作者态度 | 使用原则 |
|---|---|---|
| N1 国情与国运 | 高度重视 | 作为理解市场、产业、时代机会的底层背景；要客观描绘，避免用好坏替代判断 |
| N2 国策 | 高度重视 | 国策是长期方向和顶级设计，应优先于单条政策和行业叙事 |
| N2 政策 | 重视但需上提 | 政策多变，必须回到国策和战略方向理解，避免因政策利好产生交易冲动 |
| N3 产业 | 谨慎重视 | 产业承接国策政策，但专业门槛高，需要向上服从宏观、向下验证企业 |
| N3 行业 | 谨慎参考 | 行业景气和周期需要放在产业、政策与规划中理解，不能孤立判断 |
| N3 企业 | 高度怀疑后再验证 | 企业是最终承接者，但经营变化有滞后性，容易被包装成预期差和概念炒作 |
| N4 评论界 | 降权参考 | 重点看立场、动机和事实依据，不能用观点替代事实 |
| N4 分析界 | 有条件参考 | 有原文、数据和推导链时可辅助理解，缺少依据时降权 |
| N4 新闻界 | 先核验 | 新闻只说明事件被传播，标题和快讯不能直接作为判断依据 |
| N4 吃瓜界 | 默认过滤 | 只作为情绪观察或错误库材料，不作为事实判断 |
| N5 战略 | 重视 | 用来判断长期方向和孰轻孰重 |
| N5 方略 | 重视但需落地 | 需要继续落到政策、产业和企业承接 |
| N5 策略 | 阶段性参考 | 有时效，需要结合执行结果判断 |
| N5 计略 | 谨慎参考 | 容易滑向隐学和传闻，必须核验证据 |
| N5 谋略 | 谨慎识别 | 公开阳谋可研究，无法验证的阴谋和内幕归入伪隐学 |

---

## 5. 显学与隐学识别维度

作者强调，非金融信息通常具有隐性特征，很多内容涉及少数人理解的路径、方法、利益关系和执行细节。显学同样重要，尤其是公开规划、政策文件、真实事件和阳谋路径。

| 标签 | 含义 | 典型信息 | 处理原则 |
|---|---|---|---|
| 显学 | 公开、可查、可长期跟踪的信息 | 五年规划、政策文件、会议通稿、公开项目、正式公告、历史事件 | 高权重跟踪 |
| 隐学 | 公开文本背后的执行方法、利益结构、资源调度、实际路径 | 资源安排、竞争阻力、执行节奏、地方博弈、产业链协同 | 谨慎识别，不神秘化 |
| 伪隐学 | 包装成高级信息差、内幕、神秘渠道的信息 | 小道消息、收费爆料、暗示性传闻、无法验证的内幕 | 降权或过滤 |

# 6. N1：一国——国情与国运

## N1.1 国情信息

### 参考级别

P0 核心底层

### 作者态度

高度重视。

国情是理解金融活动、产业变迁、时代机会和市场特色的底层背景。使用时要客观描绘现实状态，避免用好坏评价替代结构判断。

### 类型定义

国情信息指一个国家的发展阶段、制度结构、资源禀赋、人口结构、产业基础、市场特色和社会变迁。

它回答的是：

> 这个市场的底层环境是什么？

### 离有形之手远近

最近。国情决定有形之手为何出手、往哪里出手、用什么方式出手。

### 代表变量

- 国家发展阶段；
- 制度特色；
- 市场化与干预并存；
- 人口结构；
- 产业基础；
- 城镇化阶段；
- 技术水平；
- 社会结构；
- 历史周期率；
- 资源禀赋。

### 典型信息

- 从农业国到工业国，再到信息技术与创新驱动；
- 市场化和干预并存；
- 国内股市呈现明显历史周期率；
- 社会面貌快速变迁导致投资风格变化；
- 中国市场具有政治经济学特征。

### 识别边界

国情信息用于理解市场底层，不直接生成个股或行业买卖结论。

### 示例

“中国股市同时具有市场化和干预特征”属于 N1.1 国情信息，参考级别 P0。

---

## N1.2 国运信息

### 参考级别

P0 核心底层

### 作者态度

高度重视。

国运用于理解国家长期发展方向、创新韧性、人口质量和时代机会。它适合作为长期背景，不适合作为短线交易依据。

### 类型定义

国运信息指国家长期发展方向、国家竞争力变化、历史阶段跃迁和长期财富变迁背景。

它回答的是：

> 这个国家长期在往哪里走？

### 离有形之手远近

最近。国运决定长期目标，影响资源配置和时代机会。

### 代表变量

- 国富民强；
- 产业升级；
- 技术创新；
- 全球竞争；
- 贸易格局；
- 国家长期使命；
- 财富变迁；
- 时代浪潮。

### 典型信息

- 积贫积弱到国富民强；
- 国家发展阶段变化带来财富变迁；
- 创新驱动成为新阶段方向；
- 时代浪潮推动投资风格变化。

### 识别边界

国运信息适合做长期背景，不宜被短线交易化。

### 示例

“国策推动意味着财富变迁”属于 N1.2 国运信息，参考级别 P0。

## N1 补充：作者对国情与国运的详细看法

### N1-A. 国情与国运的关系

作者对“国情与国运”的基本关系可以整理为：

```text
国情是国运的根基
国运是制度与发展的协同
国情与国运体现人民力量与文明创造
金融活动依托于国情
金融与产业匹配，才能理解时代机会
```

在作者框架中，国情与国运属于世界观和价值观层面的信息。它们并非短线交易信号，应理解为理解 A 股长期锚、产业变迁、时代周期和个人命运选择的底层背景。

### N1-B. 国情的定义

国情是对国家现实状态的客观描绘，不能简单用好坏、对错评判。

国情信息重点描述：

- 当前处于什么发展阶段；
- 制度如何运行；
- 人口如何变化；
- 产业如何转型；
- 地域如何分化；
- 收入和分配如何变化；
- 国家主要矛盾来自发展问题，还是分配问题；
- 当前发展模式的主体与补充是什么。

识别边界：

- 国情用于理解现实条件；
- 国情不直接推出行业或个股机会；
- 使用国情信息时，应避免情绪化评价；
- 国情判断应尽量客观，不用简单价值判断代替现实描述。

### N1-C. 国情观察维度

作者给出的国情观察维度可以整理为：

| 观察维度 | 代表问题 | 示例变量 |
|---|---|---|
| 文化与文明 | 国家文明底色是什么 | 历史文化自信、文明创造、社会价值观 |
| 历史与制度 | 制度如何塑造发展路径 | 改革开放、五年计划、市场化与干预并存 |
| 地理与地缘 | 地理结构如何影响发展 | 沿海发达、西部相对贫困、人口集中东部 |
| 人口增长与结构 | 人口数量和结构如何变化 | 老龄化、少子化、年轻人、人口质量 |
| 工业与产业 | 产业基础和工业化水平如何 | 第一制造业大国、工业化进程、产业升级 |
| 教育与创新 | 人才质量和创新能力如何 | 基础教育、单科天才、通才到专才 |
| 收入与分配 | 财富分布与收入结构如何 | 两极分化、资产性中产、分配失衡 |
| 发展模式 | 经济驱动力如何变化 | 房地产驱动、高质量发展、新质生产力 |

### N1-D. 当前中国国情变量

根据作者原文，可以把当前中国国情变量整理为：

| 变量 | 作者判断 |
|---|---|
| 历史文化 | 历史文化自信强，但历史上战火纷飞，和平时期较少 |
| 发展阶段 | 当前处在少有的和平时期，国家实力和影响力持续扩张 |
| 工业化 | 全人类最庞大的工业化进程发生在中国 |
| 规划机制 | 五年计划持续推动发展方向 |
| 基本国策 | 改革开放是基本国策 |
| 地域结构 | 沿海发达，西部相对贫困，人口集中东部 |
| 人口结构 | 处在老龄化与少子化过程中 |
| 教育方向 | 从通才培养向基础教育、单科天才和专才倾斜 |
| 发展模式 | 从居民加杠杆推动房地产，转向中央加杠杆推动新质生产力 |
| 收入结构 | 新兴领域职业薪酬高于多数传统行业 |
| 分配问题 | 改革过程中存在分配失衡和利益固化 |
| 产业方向 | 汽车、半导体、人工智能、金融科技等高附加值行业承接转型成果 |
| 社保问题 | 老一辈贡献较大，但保障体系仍需改革 |
| 核心困境 | 突破中等收入陷阱，迈向中等发达国家 |
| 发展支撑 | 需要高利润、高附加值产业支撑 |

### N1-E. 国运的定义

国运是国家在发展与创新过程中体现出的：

- 底气；
- 韧性；
- 爆发力；
- 生命力；
- 人口质量；
- 创新能力；
- 突破能力；
- 长期发展潜能。

作者强调，只要国家仍处于发展过程中，就会持续诞生新的产品、服务和突破。

### N1-F. 国运识别变量

| 变量 | 判断含义 |
|---|---|
| 发展韧性 | 面对外部不确定性仍能持续发展 |
| 创新突破 | 新产品、新服务、新技术突然出现 |
| 人口质量 | 教育和人才决定长期创新上限 |
| 青年机会 | 未来发展倾向年轻人和高知识群体 |
| 产业跃迁 | 产业从低附加值转向高附加值 |
| 国家建设 | 个人参与时代发展与国家建设 |
| 金融先行 | 金融市场提前描绘未来产业变化 |

### N1-G. 金融视角下的国情国运

从金融角度看，作者把国情国运最终落到：

```text
金融与产业是否匹配
产业转型是否创造新机会
时代抛弃的方向是否出现深度下行
金融市场是否提前描绘这种变化
个人是否参与国家建设和时代发展
```

典型表达可以整理为：

- 前二十年房地产相关资产出现巨大机会；
- 后二十年高质量发展相关资产可能出现巨大机会；
- 2018 年后，外部环境变化使产业转型更明显；
- A 股波动背后，最终仍与投资人群对国家发展路径的共识有关。

### N1-H. 识别边界

国情与国运信息属于高优先级背景信息，但使用时必须注意：

1. 国情信息不能直接生成短线交易结论；
2. 国运信息不能被包装成空泛口号；
3. “国家长期向好”不能替代行业、产业、企业承接验证；
4. 产业转型带来机会，也会带来被时代抛弃的方向；
5. 金融市场可能先行描绘未来，但价格可能透支；
6. 个人命运与国运匹配，属于长期选择维度，不等于单次买卖依据。


# 7. N2：二策——国策与政策

## N2.1 国策信息

### 参考级别

P0 核心底层

### 作者态度

高度重视。

国策是长期方向和顶级设计，应优先于单条政策、行业叙事和企业故事。它用于判断长期资源配置方向。

### 类型定义

国策信息指国家长期战略方向和远景目标。它高于具体政策，决定政策为何出现、行业为何兴衰、企业为何获得时代机会。

它回答的是：

> 国家长期要达成什么目标？

### 离有形之手远近

最近。国策是有形之手的长期目标表达。

### 代表变量

- 国家长期战略；
- 五年规划；
- 远景目标；
- 国家使命；
- 贸易与全球化；
- 科技创新；
- 产业升级；
- 安全发展；
- 区域协调；
- 出海与国际竞争。

### 典型信息

- 五年规划；
- 十四五、十五五；
- 一带一路；
- 贸易与全球化；
- 科技创新驱动；
- 国策锚定家国远景。

### 识别边界

国策锚定的是长期远景，不等同于短期利润，也不能直接推出单个板块或个股机会。

### 示例

“十五五规划即将出台，未来锚定方向可能变化”属于 N2.1 国策信息，参考级别 P0。

---

## N2.2 政策信息

### 参考级别

P1 高参考

### 作者态度

重视但需上提。

政策多变，读政策要回到国策和战略方向中理解。单条政策不能直接推出行业、板块或个股机会。

### 类型定义

政策信息指在某个阶段内，为完成国策目标而推出的具体倾向、工具和执行安排。

它回答的是：

> 当前阶段如何推进长期目标？

### 离有形之手远近

较近。政策是国策在阶段内的执行表达。

### 代表变量

- 部委政策；
- 地方政策；
- 产业扶持；
- 监管安排；
- 财政工具；
- 补贴；
- 税收；
- 专项债；
- 试点工程；
- 政策执行节奏。

### 典型信息

- 出海盈利创汇政策；
- 产业扶持政策；
- 政策推动行业周期变化；
- 阶段性调控；
- 监管收紧或放松。

### 识别边界

政策需要回到国策中理解。单条政策可能只是阶段调节，不必然代表长期趋势改变。

### 示例

## N2 补充：作者对国策与政策的详细看法

### N2-A. 国策与政策的关系

作者对国策与政策的关系可以整理为：

```text
国策是顶级设计
政策是国策的实施路径
国策解决根本性、战略性问题
政策解决阶段性、区域性、实际性问题
国策具备长期性
政策具备调节性和多变性
```

因此，理解政策时，应先回到国策，看它是否服务于长期战略方向。

### N2-B. 国策识别标准

国策通常具有以下特征：

| 特征 | 含义 |
|---|---|
| 层级高 | 由更高层级制度、会议、法律、规划承载 |
| 周期长 | 解决长期问题和战略问题 |
| 方向稳 | 不因短期行业波动轻易改变 |
| 涉及广 | 影响人口、产业、教育、贸易、科技、区域等大方向 |
| 改变财富路径 | 推动长期财富变迁和产业兴衰 |

典型国策或接近国策层级的信息：

- 计划生育及其变化；
- 改革开放；
- 碳中和；
- 对外开放；
- 五年规划；
- 人口数量与人口质量变化；
- 老龄化应对；
- 高质量发展；
- 新质生产力。

### N2-C. 政策识别标准

政策通常具有以下特征：

| 特征 | 含义 |
|---|---|
| 服务国策 | 政策是国策在阶段内的执行路径 |
| 解决局部问题 | 可针对行业、地区、风险、监管、民生等 |
| 多变 | 今年的表述可能与明年不同 |
| 影响快 | 容易带来短期快速变化 |
| 阅读难 | 内容长、层级多、执行细节复杂 |
| 有利有弊 | 对不同主体影响不同 |

典型政策信息：

- 国务院或多部门联合文件；
- 行业监管政策；
- 地方发展规划；
- 产业扶持政策；
- 新国标；
- 地方文旅政策；
- 苏超、文旅等区域发展政策；
- 具体出海支持政策；
- 隐性债务置换政策。

### N2-D. 国策理解政策的方法

作者给出的核心方法是：

```text
用国策理解政策
用长期战略理解阶段变化
用根本问题理解局部工具
用连贯性和一致性过滤政策噪音
```

具体问题包括：

- 这条政策服务哪个国策？
- 它解决的是根本问题，还是局部问题？
- 它是长期方向，还是阶段性调节？
- 它是否与更高维度战略方向一致？
- 它对行业造成短期冲击时，是否仍服务长期目标？
- 它是否只是执行节奏变化，而非大方向改变？

### N2-E. 作者给出的政策理解示例

| 示例 | 作者理解方式 |
|---|---|
| 新能源汽车安全风险与新国标电池标准 | 短期剧烈影响行业，长期服务碳中和和新能源战略 |
| 固态电池炒作 | 市场在旧路径受政策影响后，寻找新的产业发展方向 |
| 计划生育从一胎到三胎 | 人口数量和质量的重大国策变化 |
| 老龄化与教育改革 | 人口结构和人口质量变化带来的长期方向 |
| 创新药行情 | 老龄化背景下的产业映射 |
| 婴幼儿相关行情 | 人口政策变化下的产业映射 |
| 出海 | 本质是全球化贸易，对应对外开放 |
| 隐性债务置换 | 当前是风险处理，未来依赖新发展驱动和经济增长 |

### N2-F. 出海政策的高维概括

作者把“出海”简化为：

```text
出海
→ 全球化贸易
→ 对外开放
→ 贸易公平与互利共赢
→ 合理价格、合规法律、优秀产品质量与服务
→ 出海品牌建立
→ 对外开放进一步深化
```

因此，“出海”不能只看短期订单或热点板块，还要看：

- 价格是否合理；
- 法律是否合规；
- 产品质量是否优秀；
- 服务是否可持续；
- 企业是否形成品牌；
- 是否符合对外开放的长期方向。

### N2-G. 普通人处理政策信息的边界

作者对普通人处理政策信息的态度偏谨慎：

1. 政策篇幅长、层级多、变化快，读不懂也正常；
2. 不应因获得“政策有利信息”就兴奋；
3. 政策信息应回到国情现状和根本问题；
4. 要看问题的根本解决办法；
5. 国策更贴近普通人的长期命运，政策可能只与局部主体有关；
6. 个人应通过切身体会理解时代变化，保持谦虚、谨慎、客观。

### N2-H. 识别边界

国策政策信息使用时应注意：

1. 国策长期稳定，政策阶段多变；
2. 单条政策不能直接等于投资机会；
3. 政策短期冲击与长期方向可能并存；
4. 政策利好需要落到产业和企业承接；
5. 政策文件需要区分层级；
6. 政策解读需要回到原文和国策框架；
7. 普通人可以不用焦虑政策细节，优先理解高维方向。

### N3-B. 产业、行业、企业的定义

| 类型 | 定义 | 示例 |
|---|---|---|
| 产业 | 多个相关行业组成的完整产业链，覆盖从原材料到最终消费的全过程 | 电子信息产业包括芯片设计、制造、设备生产、软件开发、终端销售 |

### N3-C. 三业归纳顺序

作者明确提出，归纳这类信息时应采用自上而下顺序：

1. 产业承接国家目标；
2. 行业体现产业内部的分工和周期；
3. 企业是最终经营主体；
4. 企业机会需要服从产业方向；

```text
带着怀疑心态，总比盲目好。
```

这句话在字典中应作为识别边界，而并非流程动作。它提醒使用者：产业、行业、企业信息即使看起来正确，也需要回到宏观、政策、经营和价格反应中继续核验。


# 9. N4：四界——评论界、分析界、新闻界、吃瓜界

四界信息主要影响认知、情绪、话语权和传播路径。它们有好有坏，目的各不相同，需要识别立场、动机、对象和事实依据。

## N4.1 评论界信息

### 参考级别

P3 传播参考；情绪化时降为 PX

### 作者态度

降权参考。

评论界信息重点看立场、动机、受众和事实依据。缺少事实支撑时应降权。

### 类型定义

评论界信息指带有明显观点、立场、价值判断和议题设置的信息。

它回答的是：

> 谁在解释这件事，想让受众形成什么判断？

### 代表变量

- 立场；
- 观点；
- 议题设置；
- 价值判断；
- 受众导向；
- 情绪动员。

### 典型信息

- 大V评论；
- 社论；
- 专栏观点；
- 政策评论；
- 行业评论；
- 舆论争论。

### 识别边界

评论界信息必须检查事实依据和发布动机。没有事实支撑的评论降权。

---

## N4.2 分析界信息

### 参考级别

P3 传播参考；有数据和原文依据时可提升为 P2 辅助

### 作者态度

有条件参考。

有原文、数据和完整推导链时可辅助理解；缺少依据、只给结论或带有商业动机时应降权。

### 类型定义

分析界信息指机构、研究员、专家、作者对国情、政策、产业、企业的解释和推导。

它回答的是：

> 这套解释链条是否成立？

### 代表变量

- 分析框架；
- 推导链条；
- 研究报告；
- 产业图谱；
- 数据解释；
- 政策解读。

### 典型信息

- 券商研报；
- 专家解读；
- 产业链分析；
- 政策分析；
- 作者框架性文章。

### 识别边界

分析界信息应检查原始依据、推导链条、数据质量和利益动机。不能用结论替代事实。

---

## N4.3 新闻界信息

### 参考级别

P3 传播参考；若为官方发布原文可另归 N2

### 作者态度

先核验。

新闻说明事件被传播，标题、快讯和二次转述不能直接作为判断依据。需要回到原文、公告、政策文件或真实事件核验。

### 类型定义

新闻界信息指媒体报道、采访、标题、快讯和事件传播。

它回答的是：

> 这件事被如何报道、如何传播？

### 代表变量

- 报道来源；
- 标题措辞；
- 事实细节；
- 传播速度；
- 媒体立场；
- 事件时间。

### 典型信息

- 媒体快讯；
- 财经报道；
- 行业新闻；
- 政策新闻；
- 事件采访。

### 识别边界

新闻界信息需要回到原文、政策文件、公告或真实事件核验。标题不能直接作为判断依据。

---

## N4.4 吃瓜界信息

### 参考级别

PX 降权 / 过滤

### 作者态度

默认过滤。

吃瓜界信息主要反映情绪刺激和传播噪音，只适合作为情绪观察或错误库材料。

### 类型定义

吃瓜界信息指传闻、八卦、群聊、截图、情绪化传播和缺乏可验证来源的信息。

它回答的是：

> 市场情绪被什么东西刺激？

### 代表变量

- 传闻；
- 群聊截图；
- 情绪；
- 阴谋论；
- 未核实消息；
- 戏剧化叙事。

### 典型信息

- 群聊传闻；
- 吃瓜截图；
- 无来源爆料；
- 情绪化段子；
- 阴谋论式解释。

### 识别边界

吃瓜界信息不作为事实判断，只能作为情绪观察。无法验证时应降权或过滤。

---

# 10. N5：五略——战略、方略、策略、计略、谋略

五略属于理解复杂信息时的“目标达成方式”和“权重判断层”。

它帮助判断：

- 孰轻孰重；
- 长期目标与短期动作的关系；
- 明面路径与隐性路径的关系；
- 目标、路径、工具、执行、博弈之间的关系。

## N5.1 战略信息

### 参考级别

P1 高参考

### 作者态度

重视。

战略信息用于判断长期方向和孰轻孰重，但不能直接翻译成短线交易机会。

### 类型定义

战略信息指最高层级的长期目标、方向选择和全局安排。

它回答的是：

> 大方向是什么？

### 代表变量

- 国家战略；
- 长期方向；
- 全局目标；
- 国际竞争；
- 安全与发展；
- 全球化布局。

### 典型信息

- 贸易与全球化；
- 科技创新驱动；
- 国家长期远景；
- 产业升级战略。

### 识别边界

战略信息是高层目标，不应直接翻译成短线交易机会。

---

## N5.2 方略信息

### 参考级别

P1-P2

### 作者态度

重视但需落地。

方略说明实现战略的总体路径，需要继续落到政策、产业和企业承接上验证。

### 类型定义

方略信息指为了实现战略而形成的总体路径和行动方针。

它回答的是：

> 通过什么路径实现战略？

### 代表变量

- 总体路线；
- 区域布局；
- 产业布局；
- 资源安排；
- 外贸路径；
- 国内国际双循环。

### 典型信息

- 通过出海开拓全球市场；
- 通过产业链协同完成国策目标；
- 通过规划推动行业兴衰切换。

### 识别边界

方略需要继续落到政策、产业和企业承接。

---

## N5.3 策略信息

### 参考级别

P2

### 作者态度

阶段性参考。

策略信息通常有时效，必须结合阶段目标、执行结果和现实反馈判断。

### 类型定义

策略信息指阶段性执行方法和具体组合手段。

它回答的是：

> 当前阶段怎么做？

### 代表变量

- 阶段政策组合；
- 监管节奏；
- 扶持重点；
- 资源倾斜；
- 项目推进；
- 试点安排。

### 典型信息

- 某阶段重点支持出海盈利创汇；
- 某阶段通过补贴、税收、金融支持推动产业；
- 某阶段先试点再推广。

### 识别边界

策略通常有时效，需要结合阶段目标和执行结果判断。

---

## N5.4 计略信息

### 参考级别

P3；无法验证时降权

### 作者态度

谨慎参考。

计略涉及具体时点、资源调度和局部博弈，容易滑向隐学和传闻，必须核验证据。

### 类型定义

计略信息指具体操作、时点安排、资源调度、竞争应对和局部博弈。

它回答的是：

> 具体怎么落地、怎么应对阻力？

### 代表变量

- 时间表；
- 节奏；
- 项目排序；
- 资源调度；
- 竞争阻力；
- 博弈安排；
- 局部动作。

### 典型信息

- 某项目先地方试点再全国推广；
- 某行业先扶持龙头再带动产业链；
- 某政策分阶段执行。

### 识别边界

计略信息容易滑向隐学和传闻，必须核验证据。无法验证时降权。

---

## N5.5 谋略信息

### 参考级别

P3；伪隐学时归 PX

### 作者态度

谨慎识别。

公开可验证的阳谋可以研究；无法验证的阴谋、内幕和收费暗示应归入伪隐学。

### 类型定义

谋略信息指更复杂的博弈、阳谋、隐性安排、竞争应对和长期局面设计。

它回答的是：

> 在复杂阻力和竞争中，如何达成目标？

### 代表变量

- 阳谋；
- 博弈；
- 外部阻力；
- 内部协调；
- 长期布局；
- 竞争对手反应；
- 话语权争夺。

### 典型信息

- 一带一路作为显学式阳谋；
- 国际竞争下的产业链博弈；
- 外部阻力推动内部产业协同；
- 长期布局与短期扰动并存。

### 识别边界

谋略信息不能神秘化。公开可验证的阳谋可以研究；无法验证的内幕、阴谋和收费信息应归入 PX。

---


# 12. 使用示例

## 示例一：十五五规划即将出台

```markdown
- 非金融信息类型代码：N2.1 + N2.3
- 类型名称：国策信息 + 规划执行信息
- 作者参考级别：P0-P1
- 离有形之手远近：很近
- 是否属于显学 / 隐学 / 伪隐学：显学
- 匹配到的代表变量：五年规划、未来锚定方向、阶段性目标
- 识别理由：五年规划直接体现国家阶段性目标和资源配置方向。
- 识别边界：不能直接推出某个板块买入结论，需要继续检查政策、产业和企业承接。
```

## 示例二：优秀龙头企业率先出海

```markdown
- 非金融信息类型代码：N3.3
- 类型名称：企业信息
- 作者参考级别：P2
- 离有形之手远近：中等
- 是否属于显学 / 隐学 / 伪隐学：显学
- 匹配到的代表变量：龙头企业、出海、开拓市场、经济正循环
- 识别理由：企业是国策政策目标的最终承接者之一。
- 识别边界：需要检查企业是否真实受益、是否有订单、利润、现金流和竞争力。
```

## 示例三：某大V评论政策利好某行业

```markdown
- 非金融信息类型代码：N4.1
- 类型名称：评论界信息
- 作者参考级别：P3 或 PX
- 离有形之手远近：较远
- 是否属于显学 / 隐学 / 伪隐学：取决于证据，缺少原文时偏 PX
- 匹配到的代表变量：观点、立场、议题设置、政策评论
- 识别理由：该信息是对政策的二次解释，并非政策本身。
- 识别边界：必须回到政策原文和产业承接验证。
```

## 示例四：群聊截图称某行业有内幕利好

```markdown
- 非金融信息类型代码：N4.4
- 类型名称：吃瓜界信息
- 作者参考级别：PX
- 离有形之手远近：很远
- 是否属于显学 / 隐学 / 伪隐学：伪隐学
- 匹配到的代表变量：群聊截图、未核实消息、内幕暗示
- 识别理由：无正式来源，主要刺激情绪和交易冲动。
- 识别边界：不作为事实判断，只能作为情绪观察或错误库材料。
```

# 13. 一句话总结

非金融信息类型的核心排序是：

```text
国情国运
→ 国策政策
→ 行业产业企业
→ 评论分析新闻吃瓜
→ 战略方略策略计略谋略
```

使用时，先判断一条非金融信息离“有形之手”有多近，再判断它属于目标、路径、承接、传播还是谋略层。
