# 贵州茅台资金面判断：当前能不能买

## 核心结论

**资金状态：证据不足。当前行动：新资金 `observe`，已有持仓 `review`。**

仅从资金面出发，现在不能给出有依据的 `buy`。原因不是已经确认资金流出，而是今天的融资、ETF份额、北向持仓和订单流数据均缺失，无法判断增量资金来自哪里、是否持续，以及价格是否对流入作出正向响应。数据缺失不能被解释为“没有流出”，也不能用历史印象代替当前证据。

ETF数据对贵州茅台还需进一步拆分：宽基ETF申购只能说明指数层面的被动配置需求，消费或酒类ETF的份额变化才与贵州茅台更直接相关；即使ETF申购增加，也不能单独证明主动资金看多公司。

## 新资金行动：`observe`

建议当前资金面仓位为 **0%**；如果投资决策同时有独立且充分的基本面、估值依据，试探仓也应控制在计划总仓位的 **10%以内**，且不能把它表述为资金面确认后的买入。

将行动升级为 `buy` 前，至少补齐并验证以下条件中的三项：

1. 融资余额或融资净买入由弱转强，并连续2—3个交易日改善，而非单日脉冲；
2. 北向持仓或可比外资流向显示连续增持，而非仅由股价变动造成持仓市值上升；
3. 消费、酒类或相关宽基ETF份额增加，并能确认贵州茅台获得实际被动买盘；
4. 订单流由主动卖出转为主动买入，大额成交不再持续压盘；
5. 股价对资金改善有正向响应：放量站稳关键价位，且相对沪深300和白酒板块由弱转强。

若只有价格上涨而资金数据仍缺失，维持 `observe`，不追涨。

## 已有持仓行动：`review`

资金数据缺失本身不构成 `sell`，因此不建议仅因“看不到数据”机械减仓。应先检查持仓逻辑和价格风险：

- 基本面逻辑未变、仓位不高且股价守住原定风险位：维持原仓，不加仓；
- 股价弱于白酒板块和沪深300、反弹缩量或跌破原定风险位：执行 `reduce`，将现有仓位降低 **1/3—1/2**；
- 后续补齐数据后若融资、北向和订单流多数持续流出，同时价格继续走弱：升级为 `reduce / sell`；
- 若资金来源出现多项连续改善，且价格与行业广度同步确认：再分批加仓，每次不超过计划总仓位的 **10%—15%**。

## 最小补数清单

下一次复核至少需要：数据截止时间、近1日/5日/20日融资变化、北向持股数量变化、相关ETF份额及净申购、主动买卖订单流、贵州茅台相对沪深300与白酒指数表现。应避免把不同口径的数据简单相加，也不能把大单交易直接等同于机构行为。

## 置信度与边界

**结论置信度：高（对“当前证据不足、不能据此给出资金面买点”的判断）；方向判断置信度：低。**

本报告只回答资金面是否支持现在买入，不评价贵州茅台的内在价值、估值安全边际、业绩趋势或长期配置价值。补齐数据后，结论可能升级为 `buy`，也可能转为 `reduce / sell`。
