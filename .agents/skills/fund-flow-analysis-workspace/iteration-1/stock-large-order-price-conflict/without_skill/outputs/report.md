# 结论：先不加仓，维持半仓并进入 `review`

今天的信号不是“主力流入即可买入”，而是**资金流指标与价格表现明显冲突**。在股价冲高回落、收跌 2%、高换手且近 20 日持续跑输行业的组合下，3 亿元特大单净流入不足以证明有持续性增量资金承接。对已有半仓而言，当前更合适的动作是 `review`，而不是 `buy`。

## 为什么不能把特大单净流入直接当成加仓信号

- “特大单”只是按成交单笔规模划分，不能直接等同于机构或真正的主力资金；大单拆分、对倒、被动成交等都可能造成误判。
- 资金流统计口径通常依据主动买卖方向推算，并非账户级资金身份核验；不同平台还可能给出不同结果。
- 如果 3 亿元净流入确有较强买盘意义，股价却仍冲高回落并收跌，说明上方抛压至少同样强，买盘暂未取得定价权。
- 高换手意味着筹码快速交换，但无法单独判断是吸筹还是派发。结合冲高回落，更应防范“放量但承接失败”。
- 近 20 日跑输行业说明个股相对强度仍弱，今天的单日资金指标尚未扭转中期弱势。

## 当前资金状态

判断为：**多空分歧，偏弱；证据不足以确认资金流入趋势。**

支持资金可能介入的证据是特大单净流入 3 亿元和高换手；反证是价格收跌、盘中冲高失败以及 20 日相对行业弱势。价格是资金博弈后的最终结果，因此在冲突没有化解前，应降低对单一资金流指标的权重。

## 已有半仓的行动建议

当前动作：`review`，维持约半仓，不追加强仓。

- 不因今天的 3 亿元特大单净流入直接加仓。
- 若后续 1—3 个交易日跌破今天低点，且成交量未明显收缩，或继续显著跑输行业，说明承接失败，可考虑将仓位降至约 1/4—1/3，即 `reduce`。
- 若已有明确的基本面失效条件，应优先执行原止损或减仓规则，不应让资金流数据覆盖基本面证伪。

## 何时才考虑加仓

至少等待以下信号中的多数同时出现，再把动作从 `review` 升级为小幅 `buy`：

1. 股价重新站上今天冲高回落区间的关键压力位，并能收盘保持，而非盘中短暂突破；
2. 上涨日成交量温和放大、回调日缩量，价格开始对流入作出正向响应；
3. 连续 3—5 日的资金方向仍偏流入，而不是只有单日特大单数据突出；
4. 相对行业强度止跌并转强，至少不再持续跑输；
5. 市场广度、所属板块和公司基本面没有同步恶化。

届时也只建议分批增加约 1/10—1/6 仓位，使总仓位先升至约 60%—65%，而不是一次性满仓；加仓后的失效条件可设为重新跌回突破位下方或再度放量冲高回落。

## 下一验证点

重点观察未来 1—3 日：今天低点能否守住、冲高区域能否被收复、量价是否改善、资金流是否具备连续性，以及个股相对行业是否止弱。若这些信号没有改善，今天的“特大单净流入”更可能只是分歧成交，而不是可执行的加仓依据。

置信度：中等。由于缺少股票代码、3 亿元占成交额和流通市值的比例、资金流统计来源、关键价位及基本面信息，结论应保持条件化。
