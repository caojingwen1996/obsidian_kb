## 6. 资金分歧与传导链

- 主导链条：当日大额主动交易代理为正 → 盘中形成上冲 → 抛压和卖盘承接了买盘 → 收盘下跌且相对行业中期仍弱 → 当前状态为`多空分歧`。
- 反向链条：潜在套牢盘、短线获利盘或其他未知卖盘 → 高换手中释放筹码 → 上冲失败、收盘转弱 → 后续若流入中断，可能继续测试当日低点或前期支撑。
- 谁在承接：只能确认 F5 统计意义上的大额主动交易代理；账户身份未验证，不能写成机构、主力或产业资本承接。
- 潜在退出路径：若大额交易是短期博弈而非长期配置，价格未转强时可能快速撤出；高换手后的筹码松动、杠杆平仓或潜在减持会削弱后续承接，但相关数据均待核验。

### 核心反证

- 若后续股价迅速收复当日冲高区域、相对行业转强，并出现连续3—5日独立资金来源流入，则“买盘未取得定价权”的判断将被削弱。
- 若高换手实际处于历史低位或3亿元只占极小成交比例，当前对交易活跃度和 F5强度的判断需要下调；具体比例待核验。

### 数据缺口

- 缺少股票代码、具体交易日、F5数据平台、阈值、公式、3亿元占成交额和流通市值比例。
- 缺少近5日和近20日 F5序列、F3融资数据、F7覆盖情况以及外资、产业资本、长期资金数据。
- 缺少具体换手率、成交量历史分位、行业广度、关键价位、解禁、减持、回购和套牢盘信息。
- 缺少持仓成本、原计划、风险预算和基本面失效条件，因此不能判断半仓是否已超过用户可承受风险。

## 9. 下一验证点与失效条件

- 下一验证时间：未来1—3个交易日先做价格响应复核，满5个交易日后检查资金连续性。
- 下一验证指标：当日低点和冲高区域能否守住/收复；F5是否连续且口径可比；融资净买入和余额方向；相对行业超额收益是否止跌转正；上涨日是否放量、回调日是否缩量；解禁、减持和回购公告。
- 当前判断失效条件：价格有效收复冲高区域并持续站稳、近5日相对行业转强，同时至少一种与 F5相对独立的资金来源持续流入。
- 行动升级条件：`observe → buy` 需要资金至少达到结构性流入，且价格、相对强弱、连续性与风险预算共同确认；首笔仅用计划仓位的10%—30%。
- 行动降级条件：已有持仓若跌破原计划失效位或当日关键低点，且放量、相对行业弱势扩大或多日资金转负，则 `review → reduce`；若原持仓逻辑确认失效且承接持续恶化，再 `sell`。
