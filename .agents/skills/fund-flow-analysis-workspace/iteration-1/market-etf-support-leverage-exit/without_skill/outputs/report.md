# 最近5个交易日A股资金面判断

## 核心结论

**资金状态：多空分歧，属于“ETF托底与风险资金撤退并存”，不是共振做多。**

- 融资净买入连续为负：杠杆资金持续降低风险暴露，反映交易型资金风险偏好走弱。
- 特大单5日中有4日流出：大额主动资金总体偏卖出。特大单只能作为大额交易代理，不能直接等同于机构资金。
- 宽基ETF连续申购：存在逆势承接和指数层面的稳定力量，但ETF申购者身份未知，不能直接认定为“国家队”；申购也可能来自套利、配置或短期托底需求。

三类信号没有同向：ETF承接的是指数和流动性下限，融资与特大单却在撤退。因此当前更像“有人托住指数、有人降低仓位”，而非增量资金共同入场。也不能把融资、特大单和ETF流量简单相加为全市场净流入。

## 行动建议

### 新资金：`wait / observe`

暂不把宽基ETF连续申购直接解释为买点，建议总体仓位保持 **0%—20%试探仓**，不追涨；若没有明确标的优势，可保持空仓观察。

转为 `buy` 至少需看到以下信号中的两项连续改善：

1. 融资净买入由负转正，且不是单日脉冲；
2. 特大单由持续流出转为连续净流入；
3. ETF申购后指数不再仅靠少数权重股维持，市场广度、成交额和中小盘同步改善；
4. 指数回踩时缩量、关键支撑不破，随后放量上涨。

若仅ETF继续申购，而融资和特大单仍流出，维持 `wait`，不要因“托底”而主动提高仓位。

### 已有持仓：`review / reduce`

- 对强于指数、放量上涨且有基本面催化的持仓：可保留核心仓，整体控制在 **40%—60%**，不因ETF托底加仓。
- 对弱于指数、反弹缩量、放量下跌或跌破自身关键支撑的持仓：执行 `reduce`，降至 **20%—40%**；个股自身逻辑被证伪时应进一步退出，不能等待指数托底解套。
- 若融资持续净卖出、特大单继续流出，同时ETF申购也减弱或指数跌破近5日托底区间：将其视为托底失效，仓位进一步降至 **0%—20%**。
- 若融资和特大单同步转正，且上涨从权重扩散到多数行业与个股：再将 `review` 升级为分批 `buy`，而非一次性满仓。

## 下一验证点

重点观察未来2—3个交易日：ETF申购是否真正带来价格企稳和市场广度改善，以及融资、特大单是否停止撤退。若ETF越买、市场广度越差，说明托底只能稳定指数表面，内部风险仍在累积。

## 置信度与边界

**置信度：中等。** 判断严格基于题目给出的方向性信息。由于缺少净流入金额、历史分位、指数涨跌、成交额、行业广度和ETF份额变化明细，当前可以判断资金结构，但不足以确认具体指数点位或个股买卖时机。
