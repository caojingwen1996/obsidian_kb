# A股创新药板块资金面判断

## 核心结论

- 资金状态：**结构性流入，尚未形成全面共振**。
- 空仓行动：**buy（试仓），不宜一次性重仓**。
- 建议仓位：先买计划总资金的 **10%—15%**；在广度和融资资金进一步确认前，板块总仓位暂不超过 **20%**。
- 置信度：**中等**。现有信息足以支持小仓试错，但不足以支持趋势性重仓。

## 证据拆解

| 观察项 | 信号 | 含义 |
|---|---|---|
| 近5日相关ETF份额增加 | 偏多 | 有持续配置型资金进入；但ETF申购者身份未知，不能直接视为长期机构资金 |
| 板块成交额放大 | 偏多 | 资金关注度和交易活跃度提升，但需防止放量分歧 |
| 跑赢沪深300 | 偏多 | 流入已经获得相对价格响应，不只是“有资金、没涨价” |
| 融资余额仅小幅回升 | 弱偏多 | 杠杆资金尚未明显跟进，行情风险偏好没有充分扩散 |
| 个股上涨广度一般 | 中性偏弱 | 上涨可能集中于少数权重股或核心标的，板块内部赚钱效应不足 |

ETF份额、成交额和相对收益三项共同转强，说明这不是纯粹的无效流入；但融资与上涨广度没有同步增强，因此更接近“核心资产驱动的结构性流入”，而不是整个创新药板块的共振主升。

## 空仓执行方案

当前可用 **10%—15%** 仓位试仓，优先选择流动性较好、成分分散的创新药ETF，或已有基本面研究支持的核心标的。不要因为近5日资金转强就直接打满仓位。

后续按条件调整：

- **加仓至20%—30%**：ETF份额继续增加，同时上涨广度明显改善，且融资余额连续回升；最好伴随板块放量上涨而非放量滞涨。
- **维持10%—15%观察**：ETF仍流入、板块仍有相对收益，但融资和广度继续偏弱。
- **停止买入或退出试仓**：ETF份额由增转降，板块放量却跑输沪深300，或核心权重上涨但多数个股持续下跌。

## 下一验证点

未来3—5个交易日重点检查：ETF份额增量是否持续、上涨家数占比是否扩大、融资余额是否从“小幅回升”转为连续增长，以及板块相对沪深300是否继续走强。若只有ETF流入而价格、广度开始恶化，应把它理解为承接或托底，而不是新的买入确认。

以上仓位按可投资资金比例计算，不含个人风险承受能力、持有期限和具体估值信息；若单笔可承受亏损较低，应取区间下限。
