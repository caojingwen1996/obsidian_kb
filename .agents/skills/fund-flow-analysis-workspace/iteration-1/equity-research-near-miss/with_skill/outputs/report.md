# 路由决定

**不应由 `fund-flow-analysis` 作为主技能执行。** 该请求应路由到 `bbxm-equity-research`。

## 理由

用户的主动作是完整公司研究，明确要求十年财务、公司治理、竞争格局、DCF、公允价值和目标价。这些属于基本面研究与估值任务。`fund-flow-analysis` 的路由边界明确规定：遇到“完整公司研究、十年财务、DCF、公允价值或目标价”时，应使用 `bbxm-equity-research`，资金面仅作为其中一个输入。因此，即使分析对象是个股，也不能仅凭“个股”触发资金面技能。

## 资金面 Skill 的边界角色

如完整研究需要补充交易层面的资金证据，`fund-flow-analysis` 可作为从属模块，提供：

- 贵州茅台相关杠杆资金、特大单、ETF 申赎及其他可得资金来源的方向、持续性与分歧；
- 资金流与价格、成交、相对强弱、筹码供给的交叉验证；
- 资金状态及条件化的 `buy / sell / reduce / wait / observe / review` 辅助判断。

它不负责十年财务重构、治理质量判断、竞争格局研究、DCF 建模、公允价值计算或目标价形成，也不能以资金流结论替代基本面估值结论。
