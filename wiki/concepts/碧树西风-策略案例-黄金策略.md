# 碧树西风-策略案例-黄金策略

## 1. 一句话定义

这是[[people/碧树西风|碧树西风]]用黄金举例说明的一套规则化交易案例：入场前先定义观察周期、入场锚、止损距离、盈亏比和移动止盈方式，然后只赚这套结构能够覆盖的那一段趋势收益。

它不是“黄金一定会涨”的方向判断，也不是单独的买卖建议，而是一个用黄金说明[[concepts/碧树西风-rule-交易结构拆解|交易结构拆解]]、[[concepts/碧树西风-rule-策略启动前执行|策略启动前执行]]和移动风控的案例。

## 2. 案例来源

该策略来自[[people/碧树西风|碧树西风]]在[[sources/articles/2025-09-09-碧树西风：很多人只能投资房子，能理解他们的处境么？|2025-09-09《很多人只能投资房子，能理解他们的处境么？》]]中对黄金持仓的举例。

已有的[[concepts/碧树西风-concept-移动止盈|止盈]]概念页主要沉淀固定规则、移动止损和移动止盈的通用含义；本页主要抽取其中的黄金策略结构，作为可复用案例。

## 3. 策略结构

| 模块 | 结构 |
|---|---|
| 交易对象 | 黄金。原文没有限定具体品种、合约或交易工具。 |
| 观察周期 | 周线。周线信号更少，但更不容易被日内或短期波动清洗出局。 |
| 入场锚 | 周线布林中轨附近做多。 |
| 初始止损 | 按预期盈亏比倒推。若目标收益记为 `T`，3:1 盈亏比下初始风险单位为 `R = T / 3`。 |
| 风控位置 | 入场后在布林中轨下方超过既定止损距离处设止损。 |
| 移动机制 | 随时间推移，周线布林中轨抬升，止损线被动上移。 |
| 出场方式 | 初始止损逐渐演化为移动止损，盈利后进一步变成移动止盈。 |
| 收益来源 | 不是赚所有波动，而是赚布林中轨持续抬升所允许保留下来的趋势段。 |
| 执行要求 | 入场后不需要反复盯盘、猜测、重新判断和被短期盈亏牵引。 |

### 3.1 3:1 盈亏比下的入场位与止损位

在原文举例中，3:1 盈亏比不是把止损距离固定为某个绝对数字，而是用比例关系倒推出交易结构。可把预期目标收益记为 `T`，初始风险单位记为 `R`，则：

```text
T : R = 3 : 1
R = T / 3
```

| 项目 | 示例结构 |
|---|---|
| 入场位 | 周线布林中轨附近做多。若当期周线布林中轨为 `X`，则入场位可近似记为 `X`。 |
| 目标收益 | 目标收益记为 `T`，这是 3:1 盈亏比中的“3”。 |
| 初始风险 | 初始风险记为 `R`，且 `R = T / 3`，这是 3:1 盈亏比中的“1”。 |
| 初始止损位 | `X - R`。也就是在周线布林中轨下方一个风险单位处设置止损。 |
| 初始交易结构 | 入场位 `X`，止损位 `X - R`，目标收益空间约为 `3R`。 |
| 后续调整 | 若周线布林中轨从 `X` 抬升到 `X1`，止损位也随之抬升到 `X1 - R`，从初始止损逐步演化为移动止盈。 |

如果用原文中的数字举例：目标收益约 150 美元，则风险单位约为 50 美元；但这里的 50 美元只是 `T / 3` 的示例结果，不是策略本身固定要求的止损距离。

因此，这个案例中的关键不是“黄金一定要涨到某个固定价格”，也不是“永远止损 50 美元”，而是：入场时先确定目标收益与风险单位的比例，再把止损位放在入场锚下方一个风险单位的位置。

### 3.2 目标收益 T 怎么定

目标收益 `T` 不能从“我想赚多少”出发，而要从交易结构本身出发。更稳妥的顺序是：

```text
先判断上方可见空间 T
再判断下方结构失效距离 R
最后检查 T / R 是否达到 3
```

在这个黄金案例中，`T` 可以理解为：从周线布林中轨入场后，若趋势继续，价格在触及重要压力、趋势衰竭或移动止盈前，理论上可能留下的那段可捕捉空间。

常见判断来源包括：

| 判断来源 | 含义 |
|---|---|
| 前高或重要压力区 | 如果上方很快就是强压力，目标空间 `T` 可能不足。 |
| 周线趋势斜率 | 布林中轨持续抬升，说明趋势结构仍在给移动止盈留空间。 |
| 波动区间 | 若近期周线波动空间很窄，目标收益不能随意放大。 |
| 趋势完整性 | 若价格仍沿周线结构上行，`T` 可以由后续移动止盈动态捕捉，而不是一次性固定死。 |
| 策略失效点 | 如果入场点到结构失效点太远，`R` 过大，即使方向正确也可能不满足 3:1。 |

因此，3:1 的判断不是：

```text
我想赚 150，所以我亏 50 就走
```

而是：

```text
这笔交易上方是否至少有 3R 的可见空间？
如果没有，就算方向看多，也不满足这套策略的入场条件。
```

如果采用移动止盈，`T` 不一定是一个提前写死的卖出价，而可以是入场前对“至少值得做”的空间判断；真正退出则由周线布林中轨抬升后的移动止盈完成。

## 4. 核心内涵

1. 策略在入场前已经决定“赚哪段钱”。

   入场点、止损点和目标盈亏比决定了这套策略只能赚某一类钱。中途价格怎么波动，不应该重新定义策略目标。

2. 初始止损会随趋势演化为移动止盈。

   如果入场后趋势延续，周线布林中轨逐步抬升，原先保护本金的止损线会逐步变成保护利润的移动止盈线。

3. 周线与日线的差异不是“谁更正确”，而是交易结构不同。

   周线机会少、止损少、持仓时间长，更可能吃到大段趋势；日线机会多，但止损和移动止盈触发也更多，结果更容易分化。

4. 固定模式不是保证盈利，而是把交易从情绪反应变成可定义行为。

   碧树西风在原文中用“赌场”和“赌徒”的差异说明：有固定模式时，至少知道自己依据什么下注、承受什么风险、目标是哪段收益；没有固定模式时，容易只跟随价格和情绪行动。

5. 持仓过程的核心不是预测每一次回撤，而是遵守已经定义好的结构。

   张三式执行是理解策略后让规则工作；李四式执行是价格每波动一段就重新计算收益、恐惧和兴奋，从而破坏规则。

## 5. 张三与李四的执行差异

在原文语境中，“张三”和“李四”不是具体人物画像，而是两种执行状态：

| 类型 | 执行状态 | 结果倾向 |
|---|---|---|
| 张三式执行 | 理解自己赚的是规则内的钱，入场后接受中间波动。 | 更容易把移动止盈机制持有到趋势后段。 |
| 李四式执行 | 盈亏每变化一段就重新产生强烈情绪，并试图临场处理。 | 容易在规则生效前提前离场，或在不该加减仓时动作变形。 |

这个对比强调的是[[concepts/碧树西风-rule-仓位承受力|仓位承受力]]与执行纪律：如果仓位大到让人无法承受策略内正常波动，策略即使结构正确，也会在执行层失效。

## 6. 策略边界

- 这不是对黄金未来走势的确定性预测。
- 原文没有给出具体布林参数、交易品种、手续费、滑点、保证金和换月规则。
- 本页没有对该策略进行历史回测，因此不能把它当成已验证收益模型。
- 不同黄金工具的波动、杠杆、交易时间和强平机制不同，不能直接等同。
- 周线策略需要更长持仓时间，对资金占用和心理承受力要求更高。
- 若黄金从趋势状态转为长时间震荡，周线布林中轨策略可能连续被触发止损或移动止盈。

- 原文明确把入场锚放在“周线的布林中轨”。
- 原文用目标收益 150 美元、止损 50 美元作为示例，说明 3 倍盈亏比如何按比例落到具体风控距离。
