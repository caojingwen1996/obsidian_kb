# 碧树西风对收益必须和风险、成本一起核算的看法

## 1. 当前核心观点

在 [[sources/articles/2025-11-22-碧树西风：脱离风险，脱离成本谈收益，都是耍流氓|记忆承载 - 2025-11-22 文章归档]] 中，[[people/碧树西风|碧树西风]] 的核心判断是：脱离风险谈收益，本质上是在误导自己；对个人投资者而言，脱离时间、精力和学习成本谈收益，同样是在误导自己。真正值得比较的，不是绝对赚了多少，而是单位风险和单位成本下的收益质量。

## 4. 判断依据摘要

### 1. 收益必须和风险一起报告

文章直接反对“只看赚了多少”的习惯。作者用 `38%` 回撤需要 `60%` 收益才能回本的例子，强调亏损恢复是非对称的。也就是说，风险不是收益的背景变量，而是收益的计量口径本身。

### 2. 职业投资者看单位风险，个人投资者还要看单位成本

作者进一步把视角区分开：

- 职业投资者至少要看单位风险下的收益
- 个人投资者除了看单位风险，还要看单位成本下的收益

这里的成本不只是资金利息，还包括体力、精力、盯盘时间、学习新策略的投入，以及为了多抓那一点额外利润所牺牲的生活与研究空间。

### 3. 系统应该主动放弃性价比差的利润

文中用“只做开盘十分钟和收盘十分钟，其余时间不管”的旧日记举例，说明成熟系统不必把每一段利润都吃满。若用极少时间就能获取大部分利润，其余利润完全可以放弃，换回研究新算法和重构时间结构的空间。

### 4. 好投资更像账房先生，而不是机会收集癖

作者最后把这套判断落到执行风格上：投资不该是“看见机会就想抓”，而应该像账房先生一样，先算单位风险收益，再算单位成本收益。不能通过这两道核算的机会，即使表面赚钱，也不该算进成熟系统。
