# 中证红利什么时候买入收益率最高

## 1. 原始问题

> 中证红利什么时候买入收益率最高？

## 2. 当前答案

当同时满足以下两个条件时，中证红利进入重点买入区间：

```text
中证红利股息率 >= 近10年80%分位
且
股息率 - 10年国债收益率 >= 2.5%
```

这组条件的含义是：买入时点不只看中证红利自身股息率是否高，还要看它相对无风险利率是否给出足够风险补偿。若股息率虽然高，但10年国债收益率也高，权益资产的相对吸引力可能没有绝对股息率显示得那么强。

因此，这不是“某个固定日期买入”的规则，而是一个估值与利率共同决定的买入区间判断。

## 3. 分析过程

用户提供的回测对照如下：

| 策略 | 有效样本 | 平均年化 | 中位数 |
|---|---:|---:|---:|
| 峰值窗口 | 16 | 5.49% | 6.20% |
| 股息率最高 | 17 | 23.36% | 12.95% |
| 股息率最低 | 16 | -9.48% | -7.80% |
| 每月15号 | 17 | 6.35% | 1.81% |

从这组结果看：

1. “股息率最高”样本的平均年化和中位数显著高于其他策略，说明中证红利的买入收益更依赖估值/股息率位置，而不是固定日历时点。
2. “股息率最低”样本平均年化为负，说明在股息率低位买入，高股息策略反而可能面临估值透支或安全边际不足。
3. “每月15号”作为固定日期基准，平均年化为6.35%，中位数为1.81%，弱于股息率最高样本，说明机械定投不一定能捕捉中证红利最有利的风险收益窗口。
4. “峰值窗口”平均年化为5.49%，中位数为6.20%，表现相对平稳，但不如股息率最高样本突出。

## 4. 判断框架

这个问题可以拆成两个变量：

| 变量 | 作用 | 判断含义 |
|---|---|---|
| 中证红利股息率 | 衡量指数自身估值和现金分红吸引力 | 股息率越高，通常代表估值越低、分红收益越高 |
| 10年国债收益率 | 衡量无风险利率和资金机会成本 | 国债收益率越高，权益资产需要更高股息率才有吸引力 |
| 股息率 - 10年国债收益率 | 衡量权益红利资产相对债券的风险补偿 | 利差达到2.5%以上时，红利资产相对吸引力更强 |

因此，较优买入窗口不是单纯“股息率高”，而是：

```text
自身股息率高
-> 相对历史处于高分位
-> 相对10年国债仍有足够利差
-> 红利资产进入更有吸引力的风险收益区间
```

## 5. 三种触发规则

比较实用的触发方法可以分成三类：历史分位点、绝对股息率阈值、相对债券收益率。

### 5.1 用历史分位点触发

最直接的做法，是看中证红利的市值加权股息率处在近10年历史分布中的什么位置。这个方法不是等待股息率的绝对最高点，而是等待它进入历史高股息率区间。

| 信号 | 条件 |
|---|---|
| 强买入 | 股息率处于近10年80%分位以上 |
| 分批买入 | 股息率处于近10年70%分位以上 |
| 观望 | 股息率低于近10年50%分位 |

如果使用理杏仁一类数据源，可以直接观察“股息率分位点”“80%分位点值”“50%分位点值”“20%分位点值”等字段。该方法的优点是能适应指数自身历史估值区间；缺点是历史分位本身会随样本窗口变化。

### 5.2 用绝对股息率阈值触发

从用户提供的数据口径看，中证红利近年来高股息率区间大致可以分成：

- 4%以上：开始有吸引力；
- 5%以上：明显偏高；
- 6%以上：历史上比较稀有，通常对应较好买点。

对应操作可以写成：

| 股息率 | 操作 |
|---|---|
| < 4.0% | 不买或少买 |
| 4.0%-5.0% | 小额定投 |
| 5.0%-6.0% | 加大买入 |
| > 6.0% | 重点买入 |

这个规则好理解，也方便执行；但它的缺点是不同利率环境下，4%、5%、6%的含义会变化。若无风险利率很高，5%的股息率未必有足够吸引力；若无风险利率很低，4%以上的股息率可能已经具备配置价值。

### 5.3 用股息率相对债券收益率触发

红利资产本质上需要和债券收益率比较。更稳的规则是看：

```text
中证红利股息率 - 10年期国债收益率
```

可以分成：

| 股息率 - 10年国债收益率 | 含义 |
|---|---|
| < 1.5% | 红利吸引力一般 |
| 1.5%-2.5% | 有配置价值 |
| > 2.5% | 红利性价比较高 |
| > 3.0% | 历史上偏强买点 |

这一路径和本页当前答案最一致：当股息率达到近10年80%分位以上，同时股息率相对10年国债收益率利差达到2.5%以上时，可以视为重点买入区间。
