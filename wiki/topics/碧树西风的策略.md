# 碧树西风的策略

## 主题说明

本页聚合 [[people/碧树西风|碧树西风]] 关于“策略如何选择”的上位判断。

这里的“策略”不是单个标的、单次交易或某条择时信号，而是一个人在不可测环境中如何分配资源、承担风险、等待错价、控制失败成本，并把机会压成可执行系统。

从已有页面看，碧树西风的策略主线可以概括为：

> 不把投资理解为预测某个东西一定会涨，而是先判断自己适合哪张牌桌、能承受多大失败、要赚哪一段钱，再用仓位、退出、风控和等待把机会变成可重复的正期望系统。

## 策略总纲

碧树西风的策略不是单纯“价值投资”、单纯“交易投机”或单纯“量化套利”，而更像一套跨场景的决策框架：

1. 世界不可测，单次判断正确不是系统能力。
2. 先看自己能不能进入这局，再看这局是否值得参与。
3. 收益必须和成本、风险、回撤、资金期限、退出路径一起核算。
4. 真正的优势来自不确定性中的相对确定性，而不是更敢押注。
5. 仓位、止损、对冲、兜底和复盘，是把判断转成系统的工具。
6. 策略必须匹配执行者本人和这笔钱的真实承受力。

这条总纲与 [[topics/碧树西风-概率化决策|碧树西风-概率化决策]] 和 [[topics/碧树西风-投资系统建模|碧树西风-投资系统建模]] 互相支撑。

其中，[[concepts/碧树西风-rule-成熟策略风控结构|成熟策略风控结构规则]] 把“策略不是押注相信”进一步压成可检查结构：方向判断只是策略假设，成熟策略还必须提前写清盈亏比、止损执行、对冲兜底、退出路径和极端情形预案。

## 方法论底座：调查、矛盾、实践

[[sources/articles/2025-11-01-碧树西风-阿川开启下半场？个人如何打赢这场命运之战|2025-11-01《阿川开启下半场？个人如何打赢这场命运之战？》]] 使本页的上位方法更清楚：碧树西风不是先选择某个投资流派，再寻找理由证明它正确；他先用 [[concepts/碧树西风-framework-调查矛盾实践|调查矛盾实践]] 来重新定义问题。

这套底座包括三步：

1. 调查研究：先弄清真实世界的运行方式，避免把情绪、流行说法和个人愿望误当成事实。
2. 抓主要矛盾：在每个阶段找出最卡住行动的变量，暂时搁置次要矛盾。
3. 实践迭代：通过真实行动和反馈修正认知，而不是只停留在听懂、感慨或纸面推演。

进入投资后，这套方法导向一个很关键的重定义：投资不是由持仓时间、是否加杠杆或是否买股票来决定，而是由“是否懂、是否说得清、是否可重复”来决定。

## 投资与投机的重新定义

在这篇文章中，碧树西风明确反对用“长持仓 = 投资、短持仓 = 投机”“低杠杆 = 投资、高杠杆 = 投机”这类外观标签做判断。

他的定义更接近：

- 投资：能讲清入场条件、出场条件、适用范围、赚哪一段钱，并且能在这个框架内大量重复。
- 投机：讲不清自己为什么赚钱、为什么亏钱，也不知道当前行为能否重复。

所以，一个长期持有但说不清底层逻辑的人，仍可能是在投机；一个高杠杆、短持仓但能讲清对手盘、价格错配、退出路径和保证金兜底的人，反而可能是在投资。

这一定义把策略判断从表面标签压回系统：关键不是“看起来稳不稳”，而是“是否掌握确定性、是否知道自己赚什么钱、是否能在失败时解释并修正系统”。

## 投资的主要矛盾：赚取确定性

这篇文章也补强了本页此前对“赚取确定性”的理解。

碧树西风认为，人们之所以抬高投资、鄙薄投机，底层不是因为投资一定赚钱，而是因为人们渴望确定性。钱本身只是结果之一，更重要的是生命主动权、规则可控性和能说得清自己为何得失。

因此，在他的策略框架里：

- 赚钱不是唯一主要矛盾；
- 赚取确定性才是主要矛盾；
- 投资系统的价值在于把不确定世界中的一小段可理解、可重复、可退出的机会抓出来；
- 当逻辑被市场验证时，钱是附带结果，真正被确认的是系统对世界的解释力和控制力。

这也解释了为什么他会把后文的“押极值”放在确定性框架中理解：押极值不是追刺激，而是找到自己更能解释、更能等待、更能控制失败成本的那一段机会。

## 押注极值与守株待兔

[[sources/articles/2025-11-01-碧树西风-阿川开启下半场？个人如何打赢这场命运之战|2025-11-01《阿川开启下半场？个人如何打赢这场命运之战？》]] 补充了一个更具体的策略表达：从“押注中间值历史数据”转向“押注极值”。

这里的中间值策略，接近传统长期价值投资的想象：找到好篮子，把大部分鸡蛋放进去，长期看护，通过均值、现金流、经营质量和历史规律获得收益。

但在碧树西风这段表达里，强链接时代会改变这套策略的风险结构：

- 局部风险更容易放大成全局性黑天鹅；
- 行业、平台、政策、流动性和情绪冲击可能互相传导；
- 稳定耕种式投入可能遇到颗粒无收、血本无归的尾部结果；
- 恐慌和混乱会放大错单、错价、踩踏和被迫交易。

因此，他提出的策略不是把全部资源投入长期耕种，而是：

> 只拿出小部分资源赌新篮子和极端错配，大部分资源保持等待、流动性和生存能力。

“森林里打坑挖陷阱”的比喻，表达的是提前布置低成本、高赔率机会，等待系统性混乱、行业突变、市场恐慌或错单出现时捕捉异常收益。

## 这不是无脑赌博

“押注极值”容易被误解成激进赌博，但从碧树西风已有体系看，它至少受到四个约束。

### 1. 仓位必须很小

片段中的“十分之一”和“十分之九”说明，进攻资源只是小部分，大部分资源用于保命、等待和维持机动性。

这与 [[concepts/碧树西风-rule-仓位承受力|仓位承受力规则]] 一致：仓位不是收益欲望决定的，而是由失败后能否继续生活、继续等待、继续执行决定的。

### 2. 机会必须低成本

守株待兔式策略的前提，是等待成本不能持续吞噬本金。

如果一个所谓极值机会需要长期付费、持续杠杆、承受高利息或被迫补仓，它就不再是“挖陷阱”，而变成了把粮食继续撒下去。

### 3. 收益来源必须说清

极值策略赚的不是情绪口号，而是错价、错单、流动性踩踏、对手盘失误、规则套利或风险错配。

这需要回到 [[concepts/谁负债赚谁的钱|谁负债赚谁的钱]]：谁在混乱中付款，谁承担负债，谁因为规则、情绪或流动性错误把机会让出来。

### 4. 退出路径必须存在

捕捉极端错价之后，还要能退出、能兑现、能复盘。

如果一个机会只有账面价格，没有成交、出金、对手盘、平账或退出路径，就不符合 [[concepts/碧树西风-rule-资产准入与退出路径|资产准入与退出路径规则]]。

## 与传统长期模式的差异

| 维度 | 押中间值模式 | 押极值模式 |
|---|---|---|
| 核心假设 | 历史中间值、长期经营质量和均值回归仍然可靠 | 强链接时代会放大黑天鹅、错价和尾部机会 |
| 资源使用 | 较多资源进入少数长期篮子 | 大部分资源等待，小部分资源布置极端机会 |
| 风险来源 | 看错企业、估值、周期或长期现金流 | 等不到机会、机会成本、误把高风险当低成本 |
| 收益来源 | 长期复利、经营改善、估值修复 | 错价修复、错单、流动性混乱、规则错配 |
| 关键能力 | 研究、持有、理解企业和现金流 | 等待、识别错配、控制仓位、确保退出 |
| 失败边界 | 标的长期恶化或资金无法等待 | 陷阱成本过高、频繁下注、退出路径不清 |

因此，“押极值”不是比“押中间值”天然高级，而是适用于他自认为更能执行、更符合强链接时代结构、更匹配自身资源条件的策略版本。

## 适用条件

这个策略更适合以下情形：

- 执行者不具备长期重仓研究少数优质资产的条件；
- 资金规模较小，重仓长期复利的现实意义有限；
- 有能力等待低成本机会，而不是频繁下注；
- 能识别错价、错单、规则套利或流动性错配；
- 能在入场前写清仓位、退出、止损和失效条件；
- 大部分资源不会因为等待而被生活、债务或情绪消耗掉。

它不适合以下情形：

- 用高杠杆长期等待极端事件；
- 把每个波动都误认为黑天鹅机会；
- 把“押极值”当作逃避研究和复盘的借口；
- 没有退出路径，只靠账面浮盈自我安慰；
- 因为小仓位连续失败而不断加大赌注。
