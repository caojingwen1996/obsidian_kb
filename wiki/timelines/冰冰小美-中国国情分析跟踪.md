## 4. 核心时间线

### 2018-04 至 2018-12：2018 关键节点，撕破脸

- **节点类型**：外部竞争 / 国情国运 / 产业安全 / A 股定价锚迁移。
- **阶段位置**：旧全球化、房地产和外需扩张路径开始显露边界；安全与发展、高质量发展、科技自立和新质生产力的长期定价权重开始上升。
- **对应框架**：该节点可放进 [[people/冰冰小美|冰冰小美]] “理解国情并参与国家建设”的语境中理解：金融市场最终会反映国情国运的变化，信息的金融意义不只在短期涨跌，而在它是否改变国家竞争、产业方向、企业承接能力和资金配置方式。

#### 2018 年发生了什么

| 时间 | 事件 | 事实节点 | 金融含义 |
|---|---|---|---|
| 2018-04 | 中兴事件 | 美国商务部对中兴通讯启动拒绝令，限制美国企业向中兴供应相关产品和技术，期限接近 7 年。 | 芯片、操作系统、核心器件、上游软件工具一旦被外部卡住，企业经营可能瞬间受限；“国产替代”“自主可控”“半导体设备材料”“信创”等方向开始带有国家安全和产业生存含义。 |
| 2018-07-06 | 贸易战正式开打 | 美国对约 340 亿美元中国商品加征 25% 关税，USTR 将部分商品指向受中国工业政策支持的方向，其中包括“中国制造 2025”相关方向。 | 中美摩擦从谈判压力变成真实关税成本，直接影响出口企业利润、全球供应链布局、人民币汇率预期、外资风险偏好和 A 股风险定价。 |
| 2018-08-13 | 科技投资审查加强 | FIRRMA 签署生效，强化和现代化 CFIUS 审查机制，扩大美国对外国投资的国家安全审查范围。 | 竞争从商品贸易扩展到技术投资、并购、知识产权、数据、关键基础设施和前沿科技企业控制权，资本流动开始被国家安全逻辑约束。 |
| 2018-12-01 | 孟晚舟事件 | 华为 CFO 孟晚舟在加拿大温哥华转机时被捕，随后成为中美紧张关系中的重要事件。 | 市场更直观看到，竞争对象已经落到具备全球竞争力的中国科技企业；这与“国与国竞争，落地于企业与企业竞争”的逻辑一致。 |
| 2018 全年 | A 股风险重定价 | 上证综指全年接近下跌 25%，深成指下跌约 34%，经济放缓、贸易摩擦和去杠杆共同压制投资者情绪。 | 外部冲击已经进入股价、估值和风险偏好，2018 年不只是宏观叙事变化，也是市场价格层面的风险重估。 |

#### 为什么它是关键节点

1. **全球化逻辑变了**：2018 年以前，很多中国资产的长期叙事建立在“全球化分工 -> 中国制造 -> 出口扩张 -> 房地产与工业化共振 -> 企业盈利增长 -> 资产价格上升”上；2018 年以后，外部环境转为“贸易摩擦 -> 科技封锁 -> 供应链重组 -> 安全优先 -> 国产替代 -> 产业链自主可控”。所谓“撕破脸”，是过去可以模糊处理的矛盾开始公开化、制度化、长期化。
2. **投资主线从赚钱效率转向安全与发展**：过去看企业，更多关注利润、增速、估值、商业模式和市场空间；2018 年以后，还必须观察企业是否符合国家战略，是否解决卡脖子问题，是否具备供应链安全价值，是否承接国产替代，是否属于高附加值产业，是否能参与全球竞争。
3. **A 股定价锚发生迁移**：过去主线更像“居民加杠杆 -> 房地产 -> 城市化 -> 地产链、金融、周期、消费”；未来主线更像“中央加杠杆 -> 高质量发展 -> 新质生产力 -> 半导体、AI、汽车、高端制造、军工、金融科技”。2018 年的意义在于，它加速了第二条线的形成。
4. **风险偏好从开放红利切换到竞争红利**：同样一条产业信息，2018 年前后市场会用两套不同框架定价；出口、半导体、软件、军工、新能源车、AI 和房地产等方向的含义都随外部竞争和内部发展模式变化而重估。

#### 2018 年前后的行业解释框架变化

| 行业或方向 | 2018 年前常见视角 | 2018 年后更重要的视角 |
|---|---|---|
| 出口企业 | 全球化分工与外需扩张 | 供应链安全与全球竞争 |
| 半导体 | 成长赛道、周期景气 | 国产替代与国家安全 |
| 软件 | 企业信息化、效率工具 | 信创、自主可控 |
| 军工 | 订单周期、主题行情 | 地缘风险与安全资产 |
| 新能源车 | 消费升级、产业成长 | 制造升级与全球出海 |
| AI | 技术成长与应用想象 | 新质生产力与产业革命 |
| 房地产 | 城市化与居民加杠杆 | 旧发展模式和资产负债压力 |

#### 放进冰冰小美的信息体系里怎么理解

- **不是单纯事件史**：2018 年节点的意义，不只是贸易战、中兴、FIRRMA、孟晚舟和 A 股下跌这些事件本身，而是这些事件共同改变了“什么信息有金融意义”的判断标准。
- **金融信息要回到国情国运**：在 [[topics/冰冰小美-信息的金融意义系列|冰冰小美-信息的金融意义系列]] 中，信息不是只看价格、成交和短期情绪，还要看它是否改变一国、一策、三业、四界、五略五层结构；2018 年是这种框架被市场强行验证的节点之一。
- **国与国竞争最终落到企业承接**：外部竞争公开化后，国家战略不再停留在口号层面，而会通过产业政策、资本市场、供应链重构和企业竞争力体现出来；A 股需要观察哪些企业真正承接国家建设，而不是只看题材标签。
- **它解释了长期主线为何换锚**：如果旧锚是房地产、全球化和外需扩张，新锚就是国产替代、高端制造、科技自立、新质生产力和国家安全。2018 年以后，金融市场对“国情变化”的反应更直接，也更容易沿安全与发展的主线重新定价。

#### 后续验证窗口

- 关税、科技审查和供应链重组是否持续影响出口利润、汇率预期和外资风险偏好。
- 国产替代、半导体设备材料、信创和高端制造是否能从政策方向转化为订单、利润和全球竞争力。
- 中央加杠杆、高质量发展和新质生产力是否真正改变金融资源配置方式。
- A 股是否持续把旧房地产链的估值权重迁移到科技自立、高端制造、AI、汽车、军工和金融科技等方向。
